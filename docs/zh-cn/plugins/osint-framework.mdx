**osint-framework** 插件提供 OSINT 工具目录和调查技能——涵盖 34 个类别中的 1,064 个免费情报收集工具，映射自 [osintframework.com](https://osintframework.com)。它包括基于类别的技能、可执行的调查流水线、CLI 工具执行、OPSEC 感知工作流，以及用于跨调查关联的持久化实体图谱。

<Badge text="v1.0.1" variant="note" /> <Badge text="Security" variant="tip" />

## 安装

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## 命令

### /osint-investigate

针对任意目标运行完整的 OSINT 调查。自动检测目标类型（电子邮件、域名、IP、用户名、公司、人名）并路由到正确的类别技能。

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

在工具目录中搜索与能力查询匹配的工具。

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

按类别浏览完整的工具目录。不带参数时显示所有 34 个类别及工具数量。

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## 技能

### osint-index

顶层意图路由器。当您说"查找信息"、"调查"、"查询"、"背景调查"或描述任何开源情报任务时自动激活。根据目标类型路由到正确的类别技能。

| 目标 | 路由至 |
|--------|----------|
| 用户名或账号 | `username-recon` |
| 电子邮件地址 | `email-recon` |
| 域名 | `domain-recon` |
| IP 地址 | `ip-address-recon` |
| 人名 + 公司 | `people-search` + `business-records` |
| 公司/组织 | `business-records` + `domain-recon` |
| 恶意软件哈希 | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

工具目录浏览器。提供对完整工具注册表、类别索引、调查流水线、速率限制参考和关联引擎文档的访问。

## 代理

### osint-investigator

自主多工具调查编排器。运行完整的调查工作流——工具可用性检查、目标类型检测、顺序工具执行（被动优先）、结果聚合，以及带有置信度级别和 OPSEC 分类的结构化报告生成。

与实体图谱集成：每个工具结果在 `/tmp/osint-graph/` 中创建实体和关系，用于跨调查关联。

<Aside type="caution">
  主动侦察工具（nmap、nuclei、sherlock）会直接连接目标基础设施。OPSEC 警告钩子会在运行主动工具之前发出提醒。请始终确保您已获得授权。
</Aside>

### osint-researcher

只读目录查询代理。搜索参考文件以推荐最适合给定任务的工具。从不执行工具——仅提供推荐和说明。由类别技能内部用于工具发现。

### osint-executor

单工具 CLI 执行代理。一次运行一个工具并使用特定参数，解析输出并返回结构化结果。包含 20 多个工具的逐工具执行配方，提供精确的标志、输出格式和 jq 解析模式。

## 速率限制

所有 API 调用使用内置的 `osint_curl` 辅助工具，在 HTTP 429/503 时自动进行指数退避。关键限制：

| API | 免费限额 | 环境变量 |
|-----|-----------|---------|
| ipinfo.io | 1,000/天 | — |
| crt.sh | 60/分钟 | — |
| NVD | 5/30秒（无密钥） | `NVD_API_KEY` |
| OpenCorporates | 需要密钥 | `OPENCORPORATES_API_KEY` |
| Shodan | 1/秒 | `SHODAN_API_KEY` |
| GitHub API | 60/小时（无认证） | `GH_TOKEN` |

需要密钥的 API 在环境变量未设置时会自动跳过，并显示明确的提示信息。

## 关联引擎

该插件包含一个位于 `/tmp/osint-graph/` 的持久化 JSON 实体图谱，用于在多次调查中累积发现。

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# Add entities
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# Link them
osint_rel_add "$P" "$C" "works_at" --tool github-api

# Query: all entities within 2 hops of the person
osint_graph_query "$P" 2

# Generate report
osint_graph_report
```

**置信度评分**使用多源融合：对于每个确认来源，`confidence = 1 - product(1 - ci)`。两个置信度为 0.80 的来源 → 0.96 的组合置信度。
