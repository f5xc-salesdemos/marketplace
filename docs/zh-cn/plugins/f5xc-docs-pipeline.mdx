**f5xc-docs-pipeline** 插件为 f5-sales-demo 仓库提供文档管道架构知识。它指导内容编写，阐明多仓库构建中的配置归属，并提供基于 Docker 的本地预览。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## 安装

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## 技能

### content-author

内容编写指南，涵盖文件结构、MDX 语法规则、frontmatter 要求、图片引用以及本地 Docker 预览。在创建或编辑 `docs/` 内容、处理 MDX 文件或设置本地预览时激活。

### pipeline-navigator

文档管道架构说明，解释在四个上游仓库中应在何处进行更改：docs-theme、docs-builder、docs-control 和 docs-icons。阐明配置归属（`astro.config.mjs`、`content.config.ts`）、发布调度链以及托管文件跟踪。

<Aside type="tip">
  当您需要了解哪个仓库拥有特定的配置文件或组件时，请使用 pipeline-navigator。
</Aside>

## 命令

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

启动本地 Docker 开发服务器以预览文档。运行共享的 Docker 构建镜像，并挂载您本地的 `docs/` 目录以进行实时预览。
