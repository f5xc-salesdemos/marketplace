f5xc-salesdemos 市场提供以下插件。
每个插件都为 Claude Code 添加技能、命令和代理，
帮助处理文档、演示工作流、销售方法论、
品牌规范和开发运维等方面的工作。

## 插件包含哪些内容？

每个插件可以包含：

- **技能** — Claude 在相关场景下自动激活的上下文知识
  （例如，审查 MDX 文件）
- **命令** — 您直接调用的斜杠命令
  （例如，`/f5xc-docs-tools:review-mdx`）
- **代理** — 用于复杂任务的专业化子代理
