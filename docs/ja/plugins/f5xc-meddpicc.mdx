**f5xc-meddpicc** プラグインは、複雑なエンタープライズB2Bセールス向けの
包括的なMEDDPICCセールス資格評価・案件実行フレームワークを提供します。
MEDDPICCをチェックボックス的な作業から、エビデンスベースの資格評価、
構造化された案件レビュー、実行可能なコーチングを備えた
生きた案件実行システムへと変革します。

<Badge text="v1.0.4" variant="note" /> <Badge text="生産性" variant="tip" />

## インストール

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## MEDDPICCとは？

MEDDPICCは、複数のステークホルダー、技術的リスク、調達プロセスが
進行を阻害する可能性がある複雑なB2Bセールス向けに設計された
資格評価・案件実行フレームワークです。

| 頭文字 | 要素 | 目的 |
| ------ | ------- | ------- |
| **M** | Metrics（指標） | 顧客が期待する定量化されたビジネス成果 |
| **E** | Economic Buyer（経済的意思決定者） | 最終的な予算権限を持つ個人 |
| **D** | Decision Criteria（意思決定基準） | オプションの評価に使用される要件 |
| **D** | Decision Process（意思決定プロセス） | 評価から合意までのステップバイステップの道筋 |
| **P** | Paper Process（書類プロセス） | 調達メカニズム（法務、セキュリティ、署名） |
| **I** | Identify Pain（課題の特定） | 緊急性を生み出す切実なビジネス課題 |
| **C** | Champion（チャンピオン） | 影響力のある社内推進者 |
| **C** | Competition（競合） | すべての代替手段（ベンダー、自社開発、現状維持） |

## スキル

### meddpicc-coach

案件の資格評価、パイプラインの健全性、フォーキャスト精度、
またはいずれかのMEDDPICC要素について議論する際に自動的に起動する
包括的なMEDDPICC方法論コーチングです。エビデンスベースのガイダンス、
オペレーティングリズムの推奨事項、および詳細な参考資料へのリンクを提供します。

<Aside type="tip" title="自動起動">
  このスキルは関連する場面で自動的に起動します。スラッシュコマンドは
  不要です。
</Aside>

### deal-qualification

ギャップ分析、リスク評価、フォーキャスト推奨事項を含むスコア付き
MEDDPICCスコアカード（0&ndash;24）を生成します。各要素は検証済みの
エビデンスに基づいて0&ndash;3でスコアリングされます。

### deal-review

エビデンスの差分追跡、リスク特定、担当者名と期日が設定された
アクションアイテムを含む、構造化された週次案件点検を実施します。

### mutual-action-plan

MEDDPICC要素に紐づいた顧客向けのMutual Action Plan（相互行動計画）を
作成します。評価マイルストーン、調達ロードマップ、導入準備状況、
ステークホルダーマッピングが含まれます。

### champion-test

5つの基準（権力、個人的メリット、アクセス、情報、行動）による
評価を適用し、0&ndash;15のスコアで社内コンタクトが真のチャンピオン、
成長中のチャンピオン、コーチ、または友好的なコンタクトであるかを
判定します。

## コマンド

| コマンド | 目的 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | MEDDPICCスコアカードで案件をスコアリング |
| `/f5xc-meddpicc:deal-review [account]` | 週次案件点検を実施 |
| `/f5xc-meddpicc:champion-test [contact]` | コンタクトが真のチャンピオンかを評価 |
| `/f5xc-meddpicc:build-map [account]` | Mutual Action Planを作成 |

## エージェント

### deal-analyst

ローカルのドキュメント、会議メモ、アカウントプランをレビューして
案件の健全性を分析する読み取り専用のリサーチエージェントです。
エビデンスの引用、ギャップ分析、優先度付きのアクション推奨事項を含む
構造化されたMEDDPICC評価レポートを生成します。

## リファレンスライブラリ

プラグインには、オンデマンドで読み込まれる包括的な参考資料が含まれています：

- **MEDDPICCフレームワーク** &mdash; 全8要素の完全な定義とエビデンス要件
- **ディスカバリー質問** &mdash; 案件サイクル全体にわたる各要素の実績ある質問集
- **役割と責任** &mdash; チームロール別（AE、SE、CSM、PS、Product、Channel、Deal Desk、Legal）のRACIオーナーシップマトリクス
- **案件ステージゲート** &mdash; 次のステージに進む前に満たすべき条件（初期資格評価からクローズドウォンまで）
- **アンチパターン** &mdash; エビデンスベースの改善策を伴う10の一般的な失敗パターン
- **スコアリングルーブリック** &mdash; 要素ごとの客観的な0&ndash;3スコアリング基準
- **レビューテンプレート** &mdash; レッドフラグ指標を含む週次点検の質問セット
- **MAPテンプレート** &mdash; 評価、調達、導入ロードマップを含む顧客向けアクションプラン構造
