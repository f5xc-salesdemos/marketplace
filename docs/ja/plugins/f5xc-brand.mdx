**f5xc-brand** プラグインは、すべてのコンテンツタイプにわたってF5企業ブランドアイデンティティを適用します。45色のF5パレット、タイポグラフィルール（Neusa Next Pro Wide見出し、Proxima Nova本文）、SVG限定のロゴ使用、アイコンパック選択、WCAG AAアクセシビリティ、製品用語をカバーしています。

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## インストール

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## スキル

### brand-guardian

すべてのコンテンツ作成に対するプロアクティブなブランドアイデンティティの適用。ドキュメント、図表、プレゼンテーション、動画サムネイル、ソーシャルメディアグラフィック、Mermaidチャート、またはExcalidraw描画を作成する際に自動的に有効化されます。F5カラー、タイポグラフィ、ロゴ、アイコン、トーンオブボイス、アクセシビリティの正しい使用を保証します。

### brand-reviewer

既存コンテンツのF5ブランド準拠をレビューします。カラー、タイポグラフィ、ロゴ形式、アイコン選択、代替テキスト、コントラスト比、製品用語を監査します。VIOLATIONS（違反）、WARNINGS（警告）、SUGGESTIONS（提案）を含む構造化レポートを生成します。

### visual-content

ブランド化されたビジュアルコンテンツに対するフォーマット固有のルール。Mermaid図表、Excalidraw描画、スクリーンショット、PowerPointプレゼンテーション、YouTubeサムネイル、動画スクリプト、ソーシャルメディアグラフィックに対して、アイコンパック選択、スライドレイアウト、フォントフォールバック、プラットフォーム固有の寸法に関するガイダンスを提供します。

## コマンド

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

コンテンツのF5ブランド準拠をレビューします。レビュー範囲を指定するためにオプションのパスまたはglobパターンを受け付けます。引数なしの場合、カレントディレクトリ内のすべてのコンテンツをレビューします。
