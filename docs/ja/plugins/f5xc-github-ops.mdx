**f5xc-github-ops** プラグインは、f5xc-salesdemos リポジトリ専用の GitHub 操作自動化を提供します。イシューの作成からマージ後の監視まで、Git と GitHub のライフサイクル全体を処理し、イシュー駆動開発とプレコミットリントゲートを強制します。

<Badge text="v2.2.2" variant="note" /> <Badge text="生産性" variant="tip" />

## インストール

```
/plugin install f5xc-github-ops@f5xc-salesdemos-marketplace
```

## スキル

### workflow-lifecycle

イシューの作成、ブランチ命名規則、PR作成、エラーフィードバック付きCIポーリング、マージ後の監視、検証、タスク完了基準を処理するリポジトリガバナンスワークフロー。「commit」「push」「PRを作成」「merge」と言ったとき、またはGit操作を実行したときにアクティブになります。

### upstream-contribution

フォークベース開発のためのアップストリームコントリビューションワークフロー。フォークの検出、アップストリームの調査（イシュー、PR、CONTRIBUTING.md、CIパターン）、クロスリポジトリのイシューおよびPR作成、ローカルトラッキングイシュー管理、アップストリームCI監視、およびガイド付き解決を処理します。

## エージェント

### github-ops

すべての機械的なGitおよびGitHub操作の専用エージェント。プレコミットリントゲート、イシュー作成、ブランチ作成、ステージング、コミット、プッシュ、PR作成、エラーフィードバック付きCIポーリング、インフラ障害リトライ、マージ後の監視、ブランチクリーンアップ、およびGitHub APIを介したリポジトリ設定管理を処理します。

構造化されたステータスレポートを返します：`COMPLETE`、`PRE_COMMIT_FAILED`、`CI_FAILED`、`BLOCKED`、または `FAILED`。

<Aside type="caution">
  このエージェントはGit操作の**専用**ハンドラーです。
  このプラグインを使用するリポジトリでは、`git commit`、`git push`、`gh pr create` を直接実行しないでください。
</Aside>
