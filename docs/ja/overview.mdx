**Claude Code プラグインマーケットプレイス**は、Claude Code を新しいスキル、コマンド、エージェントで拡張するプラグインのキュレーションされたコレクションです。マーケットプレイスにより、チームはツールを共有でき、すべての開発者が手動セットアップなしで同じ機能を利用できます。

**f5-sales-demo マーケットプレイス**は、[f5-sales-demo](https://github.com/f5-sales-demo) ドキュメンテーションプラットフォーム向けに構築されたプラグインを提供します。これは、共有ビルドパイプラインを使用して Astro/Starlight サイトを GitHub Pages に公開する 20 以上のリポジトリで構成されるエコシステムです。

## 仕組み

1. **マーケットプレイスを追加** — Claude Code にプラグインソースとして登録します
2. **プラグインをインストール** — 各プラグインが Claude Code セッションにスキルとコマンドを追加します
3. **ツールを使用** — プラグインはコンテキストに基づいて自動的に有効化されるか、コマンドを直接呼び出して使用します

## 利用可能なプラグイン

| プラグイン | カテゴリ | 説明 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | 生産性 | ドキュメンテーションリポジトリ向けの MDX コンテンツ検証・レビューツール |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | 生産性 | セールスエンジニアペルソナフレームワーク — デモ実行、プレゼンテーション、Q&A、環境管理 |

## クイックスタート

マーケットプレイスを追加し、ドキュメントツールプラグインをインストールします：

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

完全なウォークスルーについては、[はじめに](/marketplace/getting-started/)をご覧ください。

## アーキテクチャ

マーケットプレイスは、1 つ以上のプラグインを含む Git リポジトリです。各プラグインは、スキル、コマンド、エージェントを単一のインストール可能なユニットにパッケージ化します。

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **マーケットプレイス** — レジストリ。検出とインストールのためのメタデータとともに利用可能なプラグインを一覧表示します。
- **プラグイン** — インストール可能なユニット。1 つ以上のスキル、コマンド、またはエージェントを含みます。
- **スキル** — タスクの内容に基づいて Claude が自動的に有効化するコンテキスト知識。
- **コマンド** — ユーザーが直接呼び出すスラッシュコマンド（例：`/review-mdx`）。
- **エージェント** — 複雑なマルチステップタスクを自律的に処理する専門サブエージェント。
