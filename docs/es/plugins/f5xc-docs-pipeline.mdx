El plugin **f5xc-docs-pipeline** proporciona conocimiento sobre la
arquitectura del pipeline de documentación para los repositorios de
f5-sales-demo. Guía la creación de contenido, aclara la propiedad
de configuración en compilaciones multi-repositorio y ofrece vista
previa local basada en Docker.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productividad" variant="tip" />

## Instalación

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## Habilidades

### content-author

Guía de creación de contenido que cubre la estructura de archivos,
reglas de sintaxis MDX, requisitos de frontmatter, referencias de
imágenes y vista previa local con Docker. Se activa al crear o
editar contenido en `docs/`, trabajar con archivos MDX o configurar
la vista previa local.

### pipeline-navigator

Arquitectura del pipeline de documentación que explica dónde
realizar cambios en los cuatro repositorios upstream: docs-theme,
docs-builder, docs-control y docs-icons. Aclara la propiedad de
configuración (`astro.config.mjs`, `content.config.ts`), la cadena
de despacho de lanzamientos y el seguimiento de archivos
gestionados.

<Aside type="tip">
  Use pipeline-navigator cuando necesite comprender qué
  repositorio es propietario de un archivo de configuración o
  componente específico.
</Aside>

## Comandos

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

Inicia un servidor de desarrollo local con Docker para previsualizar
la documentación. Ejecuta la imagen Docker de compilación compartida
con su directorio local `docs/` montado para vista previa en vivo.
