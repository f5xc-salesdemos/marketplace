El plugin **f5xc-brand** aplica la identidad corporativa de marca F5
en todos los tipos de contenido. Cubre la paleta de 45 colores de F5,
las reglas tipográficas (titulares en Neusa Next Pro Wide, cuerpo en
Proxima Nova), uso exclusivo de logotipos en SVG, selección de packs
de iconos, accesibilidad WCAG AA y terminología de productos.

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## Instalación

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## Habilidades

### brand-guardian

Aplicación proactiva de la identidad de marca para toda la creación de contenido.
Se activa automáticamente al crear documentación, diagramas,
presentaciones, miniaturas de video, gráficos para redes sociales, gráficos
Mermaid o dibujos Excalidraw. Garantiza el uso correcto de los colores, la
tipografía, los logotipos, los iconos, el tono de voz y la accesibilidad de F5.

### brand-reviewer

Revisa el contenido existente para verificar el cumplimiento de la marca F5. Audita colores,
tipografía, formato de logotipo, selección de iconos, texto alternativo, ratios de contraste
y terminología de productos. Genera informes estructurados con
VIOLACIONES, ADVERTENCIAS y SUGERENCIAS.

### visual-content

Reglas específicas por formato para contenido visual con marca. Proporciona
orientación para diagramas Mermaid, dibujos Excalidraw, capturas de pantalla,
presentaciones PowerPoint, miniaturas de YouTube, guiones de video y
gráficos para redes sociales con selección de packs de iconos, diseños de diapositivas,
fuentes alternativas y dimensiones específicas por plataforma.

## Comandos

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

Revisa el contenido para verificar el cumplimiento de la marca F5. Acepta una ruta
opcional o un patrón glob para delimitar el alcance de la revisión. Sin argumentos, revisa
todo el contenido en el directorio actual.
