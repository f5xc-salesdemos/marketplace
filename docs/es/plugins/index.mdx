El marketplace f5xc-salesdemos proporciona los siguientes plugins.
Cada plugin añade habilidades, comandos y agentes a Claude Code
que ayudan con documentación, flujos de trabajo de demostración, metodología de ventas,
marca corporativa y operaciones de desarrollo.

## ¿Qué contiene un Plugin?

Cada plugin puede incluir:

- **Habilidades** — conocimiento contextual que Claude activa
  automáticamente cuando es relevante (p. ej., al revisar archivos MDX)
- **Comandos** — comandos slash que se invocan directamente
  (p. ej., `/f5xc-docs-tools:review-mdx`)
- **Agentes** — sub-agentes especializados para tareas complejas

/plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
/plugin install f5xc-github-ops@f5xc-salesdemos-marketplace
