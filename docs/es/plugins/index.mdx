El marketplace f5-sales-demo proporciona los siguientes plugins.
Cada plugin añade habilidades, comandos y agentes a Claude Code
que ayudan con la documentación, flujos de trabajo de demostración, metodología de ventas,
identidad de marca y operaciones de desarrollo.

## ¿Qué contiene un plugin?

Cada plugin puede incluir:

- **Habilidades** — conocimiento contextual que Claude activa
  automáticamente cuando es relevante (p. ej., revisión de archivos MDX)
- **Comandos** — comandos slash que se invocan directamente
  (p. ej., `/f5xc-docs-tools:review-mdx`)
- **Agentes** — subagentes especializados para tareas complejas

## Instalación de un plugin

```
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
/plugin install f5xc-brand@f5-sales-demo-marketplace
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
/plugin install f5xc-platform@f5-sales-demo-marketplace
/plugin install osint-framework@f5-sales-demo-marketplace
/plugin install f5xc-firecrawl@f5-sales-demo-marketplace
```

recorrido completo de configuración.
