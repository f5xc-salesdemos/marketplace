Un **marketplace de plugins de Claude Code** es una colección curada de
plugins que extienden Claude Code con nuevas habilidades, comandos y
agentes. Los marketplaces permiten a los equipos compartir herramientas
para que cada desarrollador obtenga las mismas capacidades sin
configuración manual.

El **Marketplace f5-sales-demo** proporciona plugins creados para
la plataforma de documentación
[f5-sales-demo](https://github.com/f5-sales-demo) — un ecosistema
de más de 20 repositorios que publican sitios Astro/Starlight en
GitHub Pages utilizando un pipeline de compilación compartido.

## Cómo funciona

1. **Añada el marketplace** a Claude Code — esto lo registra
   como una fuente de plugins
2. **Instale plugins** desde el marketplace — cada plugin añade
   habilidades y comandos a su sesión de Claude Code
3. **Utilice las herramientas** — los plugins se activan automáticamente
   según el contexto, o puede invocar sus comandos directamente

## Plugins disponibles

| Plugin | Categoría | Descripción |
|--------|-----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | Productividad | Herramientas de validación y revisión de contenido MDX para repositorios de documentación |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | Productividad | Marco de trabajo de persona Sales Engineer — ejecución de demos, presentaciones, preguntas y respuestas, gestión de entornos |

## Inicio rápido

Añada el marketplace e instale el plugin de herramientas de documentación:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Consulte [Primeros pasos](/marketplace/getting-started/) para
un tutorial completo.

## Arquitectura

Un marketplace es un repositorio Git que contiene uno o más
plugins. Cada plugin empaqueta habilidades, comandos y agentes
en una única unidad instalable.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — el registro. Lista los plugins disponibles
  con metadatos para descubrimiento e instalación.
- **Plugin** — la unidad instalable. Contiene una o más
  habilidades, comandos o agentes.
- **Skill (Habilidad)** — conocimiento contextual que Claude activa
  automáticamente según la tarea en cuestión.
- **Command (Comando)** — un comando de barra que el usuario invoca
  directamente (por ejemplo, `/review-mdx`).
- **Agent (Agente)** — un sub-agente especializado que gestiona tareas
  complejas de múltiples pasos de forma autónoma.
