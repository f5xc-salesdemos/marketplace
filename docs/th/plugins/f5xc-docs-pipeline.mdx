ปลั๊กอิน **f5xc-docs-pipeline** ให้ความรู้เกี่ยวกับสถาปัตยกรรม
ไปป์ไลน์เอกสารสำหรับรีโปซิทอรี f5-sales-demo โดยช่วยแนะนำ
การเขียนเนื้อหา อธิบายความเป็นเจ้าของการตั้งค่าข้ามการสร้าง
แบบหลายรีโป และให้การดูตัวอย่างในเครื่องผ่าน Docker

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## ทักษะ

### content-author

คู่มือการเขียนเนื้อหาครอบคลุมโครงสร้างไฟล์ กฎไวยากรณ์ MDX
ข้อกำหนด frontmatter การอ้างอิงรูปภาพ และการดูตัวอย่างในเครื่อง
ผ่าน Docker เปิดใช้งานเมื่อสร้างหรือแก้ไขเนื้อหาใน `docs/`
ทำงานกับไฟล์ MDX หรือตั้งค่าการดูตัวอย่างในเครื่อง

### pipeline-navigator

สถาปัตยกรรมไปป์ไลน์เอกสารที่อธิบายว่าควรทำการเปลี่ยนแปลง
ที่ใดข้ามรีโปซิทอรีอัปสตรีมทั้งสี่: docs-theme, docs-builder,
docs-control และ docs-icons อธิบายความเป็นเจ้าของการตั้งค่า
(`astro.config.mjs`, `content.config.ts`) ลำดับการดิสแพตช์
รีลีส และการติดตามไฟล์ที่ถูกจัดการ

<Aside type="tip">
  ใช้ pipeline-navigator เมื่อคุณต้องการทำความเข้าใจว่ารีโปซิทอรี
  ใดเป็นเจ้าของไฟล์การตั้งค่าหรือคอมโพเนนต์เฉพาะ
</Aside>

## คำสั่ง

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

เริ่มเซิร์ฟเวอร์ dev Docker ในเครื่องเพื่อดูตัวอย่างเอกสาร
รันอิมเมจ Docker build ที่ใช้ร่วมกันโดยเมาท์ไดเรกทอรี `docs/`
ในเครื่องของคุณสำหรับการดูตัวอย่างแบบสด
