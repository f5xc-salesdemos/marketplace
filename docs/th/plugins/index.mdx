f5-sales-demo marketplace มีปลั๊กอินดังต่อไปนี้
แต่ละปลั๊กอินจะเพิ่มทักษะ คำสั่ง และเอเจนต์ให้กับ Claude Code
ที่ช่วยในเรื่องเอกสาร เวิร์กโฟลว์สาธิต วิธีการขาย
การสร้างแบรนด์ และการดำเนินการด้านการพัฒนา

## ปลั๊กอินประกอบด้วยอะไรบ้าง?

แต่ละปลั๊กอินสามารถประกอบด้วย:

- **ทักษะ** — ความรู้ตามบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติเมื่อเกี่ยวข้อง (เช่น การรีวิวไฟล์ MDX)
- **คำสั่ง** — คำสั่ง slash ที่คุณเรียกใช้โดยตรง
  (เช่น `/f5xc-docs-tools:review-mdx`)
- **เอเจนต์** — ซับเอเจนต์เฉพาะทางสำหรับงานที่ซับซ้อน

```
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
/plugin install f5xc-brand@f5-sales-demo-marketplace
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
/plugin install f5xc-platform@f5-sales-demo-marketplace
/plugin install osint-framework@f5-sales-demo-marketplace
/plugin install f5xc-firecrawl@f5-sales-demo-marketplace
```
