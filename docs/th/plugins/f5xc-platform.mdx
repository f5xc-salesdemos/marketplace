ปลั๊กอิน **f5xc-platform** ให้ระบบอัตโนมัติแบบครบวงจรสำหรับแพลตฟอร์ม
F5 Distributed Cloud ครอบคลุมการดำเนินงานบนเว็บคอนโซลผ่าน
Chrome DevTools MCP, การดำเนินงาน REST API ที่รองรับ spec ผ่าน cURL,
การยืนยันตัวตนหลายผู้ให้บริการ และการวิเคราะห์การกำหนดค่าครอบคลุม
98 ประเภททรัพยากรใน 38 โดเมน

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## ทักษะ

### api-operations

การดำเนินงาน CRUD ของ API ที่รองรับ spec ครอบคลุม 98 ประเภททรัพยากร
ใน 38 โดเมน ให้ endpoint ที่แม่นยำ, เทมเพลตเพย์โหลด,
การจัดลำดับ dependency และการประกอบเวิร์กโฟลว์หลายขั้นตอนสำหรับ
HTTP load balancer, origin pool, นโยบาย WAF, โซน DNS,
ใบรับรอง, cloud site และอื่น ๆ

### config-analysis

ความสามารถในการวิเคราะห์และให้คำแนะนำเกี่ยวกับการกำหนดค่า วิเคราะห์
การกำหนดค่า JSON ของลูกค้าเพื่อตอบคำถามเกี่ยวกับสถานะความปลอดภัย,
การเปิดใช้ฟีเจอร์, การตั้งค่าโหมด, การยกเว้น และแนวปฏิบัติที่ดีที่สุด
รองรับการถาม-ตอบแบบหลายรอบพร้อมบริบทที่บีบอัดระหว่างการส่งงาน

### api-auth

การยืนยันตัวตน API รองรับการยืนยันตัวตนด้วย API token header และ
การยืนยันตัวตนด้วยใบรับรอง P12 พร้อมขั้นตอนการตรวจสอบ token

### console-auth

การยืนยันตัวตนหลายผู้ให้บริการสำหรับเว็บคอนโซล F5 XC
ตรวจจับประเภทการเข้าสู่ระบบอัตโนมัติ: การเข้าสู่ระบบ F5 XC แบบดั้งเดิม,
Azure SSO พร้อมเซสชันที่แคชไว้ หรือ Azure SSO พร้อม DUO verified push MFA

### console-navigator

การนำทางแบบกำหนดได้ไปยังส่วนต่าง ๆ ของคอนโซล F5 XC แปลง
ชื่อส่วนเป็นเส้นทาง URL, ตรวจสอบสุขภาพเซสชัน และยืนยัน
การโหลดหน้า

### platform-index / api-index / console-index

ตัวกำหนดเส้นทางตามเจตนาที่จำแนกคำขอของผู้ใช้และมอบหมายให้
ทักษะและเอเจนต์ที่เหมาะสม สิ่งเหล่านี้ทำงานอัตโนมัติและ
ไม่สามารถเรียกใช้โดยผู้ใช้ได้

## คำสั่ง

### /check-api-token

```
/f5xc-platform:check-api-token
```

ตรวจสอบความถูกต้องของ API token ของ F5 XC โดยมอบหมายให้เอเจนต์
api-operator ดำเนินการ

### /login-console

```
/f5xc-platform:login-console
```

ยืนยันตัวตนเข้าสู่คอนโซล F5 Distributed Cloud ผ่านเอเจนต์
console-operator

## เอเจนต์

### api-operator

เอเจนต์ REST API อัตโนมัติที่รันลำดับ cURL และ jq สำหรับ
CRUD ทรัพยากร, การตรวจสอบ token และการดำเนินงานกำหนดค่า
ทำงานในซับเอเจนต์เพื่อรักษาเพย์โหลด JSON ของ API ที่มีรายละเอียดมาก
ให้อยู่นอกบริบทเซสชันหลัก

### console-operator

เอเจนต์ระบบอัตโนมัติของเบราว์เซอร์ที่รันลำดับเครื่องมือ Chrome DevTools MCP
สำหรับการยืนยันตัวตน, การนำทาง และการโต้ตอบกับฟอร์ม
ทำงานในซับเอเจนต์เพื่อรักษา snapshot ของเบราว์เซอร์ที่ใช้ token มาก
ให้อยู่นอกเซสชันหลัก

### config-analyzer

เอเจนต์วิเคราะห์การกำหนดค่าแบบอ่านอย่างเดียว วิเคราะห์การกำหนดค่า
JSON ของลูกค้าเทียบกับไฟล์อ้างอิงโปรไฟล์ทรัพยากรเพื่อตอบคำถาม
เกี่ยวกับสถานะความปลอดภัย, การเปิดใช้ฟีเจอร์, การเปลี่ยนโหมด
และแนวปฏิบัติที่ดีที่สุด

<Aside type="caution">
  เอเจนต์ config-analyzer เป็นแบบ **ชั่วคราว** &mdash; สร้างขึ้นใหม่
  สำหรับแต่ละคำถามโดยไม่มีหน่วยความจำระหว่างการส่งงาน คำถามติดตามผล
  ต้องรวมผลการค้นพบก่อนหน้าที่บีบอัดแล้วไว้ในพรอมต์การส่งงาน
</Aside>
