ปลั๊กอิน **f5xc-brand** บังคับใช้อัตลักษณ์แบรนด์องค์กร F5
ครอบคลุมเนื้อหาทุกประเภท ครอบคลุมจานสี F5 จำนวน 45 สี,
กฎตัวอักษร (หัวข้อใช้ Neusa Next Pro Wide, เนื้อหาใช้ Proxima Nova),
การใช้โลโก้แบบ SVG เท่านั้น, การเลือกชุดไอคอน, การเข้าถึงตามมาตรฐาน
WCAG AA และศัพท์เฉพาะผลิตภัณฑ์

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## การติดตั้ง

```
/plugin install f5xc-brand@f5xc-salesdemos-marketplace
```

## ทักษะ

### brand-guardian

การบังคับใช้อัตลักษณ์แบรนด์เชิงรุกสำหรับการสร้างเนื้อหาทั้งหมด
เปิดใช้งานโดยอัตโนมัติเมื่อสร้างเอกสาร, ไดอะแกรม,
งานนำเสนอ, ภาพขนาดย่อวิดีโอ, กราฟิกสื่อสังคม, แผนภูมิ Mermaid
หรือภาพวาด Excalidraw ตรวจสอบการใช้สี F5, ตัวอักษร, โลโก้,
ไอคอน, น้ำเสียงในการสื่อสาร และการเข้าถึงอย่างถูกต้อง

### brand-reviewer

ตรวจสอบเนื้อหาที่มีอยู่เพื่อความสอดคล้องกับแบรนด์ F5 ตรวจสอบสี,
ตัวอักษร, รูปแบบโลโก้, การเลือกไอคอน, ข้อความ alt, อัตราส่วนคอนทราสต์
และศัพท์เฉพาะผลิตภัณฑ์ สร้างรายงานที่มีโครงสร้างพร้อม
การละเมิด (VIOLATIONS), คำเตือน (WARNINGS) และข้อเสนอแนะ (SUGGESTIONS)

### visual-content

กฎเฉพาะรูปแบบสำหรับเนื้อหาภาพที่มีแบรนด์ ให้คำแนะนำ
สำหรับไดอะแกรม Mermaid, ภาพวาด Excalidraw, ภาพหน้าจอ,
งานนำเสนอ PowerPoint, ภาพขนาดย่อ YouTube, สคริปต์วิดีโอ
และกราฟิกสื่อสังคม พร้อมการเลือกชุดไอคอน, เค้าโครงสไลด์,
ฟอนต์สำรอง และขนาดเฉพาะแพลตฟอร์ม

## คำสั่ง

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

ตรวจสอบเนื้อหาเพื่อความสอดคล้องกับแบรนด์ F5 รับพาธหรือรูปแบบ
glob เพิ่มเติมเพื่อกำหนดขอบเขตการตรวจสอบ หากไม่ระบุอาร์กิวเมนต์
จะตรวจสอบเนื้อหาทั้งหมดในไดเรกทอรีปัจจุบัน
