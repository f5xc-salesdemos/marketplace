ปลั๊กอิน **f5xc-meddpicc** มอบกรอบการทำงานด้านการคัดกรองและดำเนินการขาย
MEDDPICC อย่างครอบคลุมสำหรับการขาย B2B ระดับองค์กรที่ซับซ้อน โดยเปลี่ยน
MEDDPICC จากการทำเช็คลิสต์ธรรมดาให้กลายเป็นระบบดำเนินการขายที่มีชีวิต
พร้อมการคัดกรองตามหลักฐาน การทบทวนดีลแบบมีโครงสร้าง
และการโค้ชที่นำไปปฏิบัติได้จริง

## การติดตั้ง

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## MEDDPICC คืออะไร?

MEDDPICC คือกรอบการทำงานด้านการคัดกรองและดำเนินการขายที่ออกแบบมาสำหรับ
การขาย B2B ที่ซับซ้อน ซึ่งมีผู้มีส่วนได้ส่วนเสียหลายฝ่าย ความเสี่ยงทางเทคนิค
และขั้นตอนการจัดซื้อจัดจ้างที่อาจทำให้โมเมนตัมสะดุดได้

| ตัวอักษร | องค์ประกอบ | วัตถุประสงค์ |
| ------ | ------- | ------- |
| **M** | Metrics | ผลลัพธ์ทางธุรกิจเชิงปริมาณที่ลูกค้าคาดหวัง |
| **E** | Economic Buyer | บุคคลที่มีอำนาจตัดสินใจด้านงบประมาณขั้นสุดท้าย |
| **D** | Decision Criteria | ข้อกำหนดที่ใช้ในการประเมินตัวเลือก |
| **D** | Decision Process | เส้นทางทีละขั้นตอนจากการประเมินไปสู่ข้อตกลง |
| **P** | Paper Process | กลไกการจัดซื้อจัดจ้าง (กฎหมาย ความปลอดภัย การลงนาม) |
| **I** | Identify Pain | ปัญหาทางธุรกิจที่เร่งด่วนและสร้างความจำเป็นเร่งด่วน |
| **C** | Champion | ผู้สนับสนุนภายในที่มีอิทธิพล |
| **C** | Competition | ทางเลือกทั้งหมด (ผู้ขายรายอื่น สร้างเอง ไม่ทำอะไร) |

## ทักษะ

### meddpicc-coach

การโค้ชวิธีการ MEDDPICC อย่างครอบคลุมที่เปิดใช้งานอัตโนมัติ
เมื่อคุณพูดคุยเกี่ยวกับการคัดกรองดีล สุขภาพไปป์ไลน์
ความแม่นยำในการพยากรณ์ หรือองค์ประกอบ MEDDPICC ใดก็ตาม
ให้คำแนะนำตามหลักฐาน ข้อแนะนำจังหวะการทำงาน
และลิงก์ไปยังเอกสารอ้างอิงโดยละเอียด

<Aside type="tip" title="เปิดใช้งานอัตโนมัติ">
  ทักษะนี้จะเปิดใช้งานอัตโนมัติเมื่อเกี่ยวข้อง — ไม่จำเป็นต้องใช้คำสั่ง slash
</Aside>

### deal-qualification

สร้างสกอร์การ์ด MEDDPICC พร้อมคะแนน (0&ndash;24) พร้อมการวิเคราะห์ช่องว่าง
การประเมินความเสี่ยง และคำแนะนำในการพยากรณ์ แต่ละองค์ประกอบ
จะได้คะแนน 0&ndash;3 ตามหลักฐานที่ได้รับการยืนยัน

### deal-review

อำนวยความสะดวกในการตรวจสอบดีลรายสัปดาห์แบบมีโครงสร้าง พร้อมการติดตาม
ความเปลี่ยนแปลงของหลักฐาน การระบุความเสี่ยง และรายการดำเนินการ
พร้อมผู้รับผิดชอบและกำหนดเวลา

### mutual-action-plan

สร้าง Mutual Action Plan สำหรับลูกค้าที่เชื่อมโยงกับองค์ประกอบ MEDDPICC
รวมถึงเหตุการณ์สำคัญในการประเมิน แผนงานการจัดซื้อจัดจ้าง
ความพร้อมในการนำไปใช้งาน และการแมปผู้มีส่วนได้ส่วนเสีย

### champion-test

ใช้การประเมิน 5 เกณฑ์ (Power, Personal Win, Access,
Intel, Action) คะแนน 0&ndash;15 เพื่อพิจารณาว่าผู้ติดต่อภายใน
ของคุณเป็นแชมเปี้ยนตัวจริง แชมเปี้ยนที่กำลังเติบโต โค้ช
หรือผู้ติดต่อที่เป็นมิตร

## เอเจนต์

### deal-analyst

เอเจนต์วิจัยแบบอ่านอย่างเดียวที่วิเคราะห์สุขภาพของดีลโดยตรวจสอบ
เอกสารภายใน บันทึกการประชุม และแผนบัญชี สร้างรายงานการประเมิน
MEDDPICC แบบมีโครงสร้างพร้อมการอ้างอิงหลักฐาน การวิเคราะห์ช่องว่าง
และคำแนะนำการดำเนินการตามลำดับความสำคัญ

## ไลบรารีอ้างอิง

ปลั๊กอินนี้มีเอกสารอ้างอิงที่ครอบคลุมซึ่งโหลดตามความต้องการ:

- **กรอบการทำงาน MEDDPICC** &mdash; คำจำกัดความครบถ้วนและข้อกำหนด
  หลักฐานสำหรับองค์ประกอบทั้ง 8 รายการ
- **คำถามในการค้นหา** &mdash; คำถามที่ผ่านการทดสอบภาคสนามสำหรับ
  ทุกองค์ประกอบตลอดวงจรการขาย
- **บทบาทและความรับผิดชอบ** &mdash; เมทริกซ์ความเป็นเจ้าของ RACI
  ตามบทบาทในทีม (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **เกณฑ์ผ่านขั้นตอนดีล** &mdash; สิ่งที่ต้องเป็นจริงก่อนที่จะเลื่อนขั้น
  (ตั้งแต่การคัดกรองเบื้องต้นจนถึง Closed Won)
- **รูปแบบที่ควรหลีกเลี่ยง** &mdash; รูปแบบความล้มเหลวที่พบบ่อย 10 รูปแบบ
  พร้อมการแก้ไขตามหลักฐาน
- **เกณฑ์การให้คะแนน** &mdash; เกณฑ์การให้คะแนนเชิงวัตถุวิสัย 0&ndash;3
  ต่อองค์ประกอบ
- **เทมเพลตการทบทวน** &mdash; ชุดคำถามตรวจสอบรายสัปดาห์พร้อม
  ตัวชี้วัดสัญญาณเตือน
- **เทมเพลต MAP** &mdash; โครงสร้างแผนดำเนินการสำหรับลูกค้า
  พร้อมแผนงานการประเมิน การจัดซื้อจัดจ้าง และการนำไปใช้งาน
