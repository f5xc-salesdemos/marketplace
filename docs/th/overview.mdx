**ตลาดปลั๊กอิน Claude Code** คือคอลเลกชันที่คัดสรรของ
ปลั๊กอินที่ขยายความสามารถของ Claude Code ด้วยทักษะ คำสั่ง
และเอเจนต์ใหม่ ตลาดปลั๊กอินช่วยให้ทีมแชร์เครื่องมือ
เพื่อให้นักพัฒนาทุกคนได้รับความสามารถเดียวกันโดยไม่ต้อง
ตั้งค่าด้วยตนเอง

**ตลาดปลั๊กอิน f5-sales-demo** มีปลั๊กอินที่สร้างขึ้นสำหรับ
แพลตฟอร์มเอกสาร [f5-sales-demo](https://github.com/f5-sales-demo)
— ระบบนิเวศของรีโพซิทอรีกว่า 20 แห่งที่เผยแพร่เว็บไซต์
Astro/Starlight ไปยัง GitHub Pages โดยใช้ไปป์ไลน์การบิลด์ร่วมกัน

## วิธีการทำงาน

1. **เพิ่มตลาดปลั๊กอิน** ไปยัง Claude Code — การดำเนินการนี้จะ
   ลงทะเบียนเป็นแหล่งปลั๊กอิน
2. **ติดตั้งปลั๊กอิน** จากตลาดปลั๊กอิน — ปลั๊กอินแต่ละตัวจะเพิ่ม
   ทักษะและคำสั่งไปยังเซสชัน Claude Code ของคุณ
3. **ใช้เครื่องมือ** — ปลั๊กอินจะเปิดใช้งานโดยอัตโนมัติตามบริบท
   หรือคุณสามารถเรียกใช้คำสั่งโดยตรง

## ปลั๊กอินที่พร้อมใช้งาน

| ปลั๊กอิน | หมวดหมู่ | คำอธิบาย |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | เพิ่มประสิทธิภาพ | เครื่องมือตรวจสอบและรีวิวเนื้อหา MDX สำหรับรีโพซิทอรีเอกสาร |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | เพิ่มประสิทธิภาพ | เฟรมเวิร์กบุคลิกภาพ Sales Engineer — การดำเนินการสาธิต การนำเสนอ ถาม-ตอบ การจัดการสภาพแวดล้อม |

## เริ่มต้นอย่างรวดเร็ว

เพิ่มตลาดปลั๊กอินและติดตั้งปลั๊กอินเครื่องมือเอกสาร:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

ดู [เริ่มต้นใช้งาน](/marketplace/getting-started/) สำหรับ
คำแนะนำฉบับสมบูรณ์

## สถาปัตยกรรม

ตลาดปลั๊กอินคือรีโพซิทอรี Git ที่ประกอบด้วยปลั๊กอิน
หนึ่งตัวหรือมากกว่า ปลั๊กอินแต่ละตัวจะรวมทักษะ คำสั่ง
และเอเจนต์เข้าเป็นหน่วยเดียวที่สามารถติดตั้งได้

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **ตลาดปลั๊กอิน** — รีจิสทรี แสดงรายการปลั๊กอินที่พร้อมใช้งาน
  พร้อมเมตาดาต้าสำหรับการค้นหาและติดตั้ง
- **ปลั๊กอิน** — หน่วยที่สามารถติดตั้งได้ ประกอบด้วยทักษะ
  คำสั่ง หรือเอเจนต์หนึ่งรายการหรือมากกว่า
- **ทักษะ** — ความรู้ตามบริบทที่ Claude เปิดใช้งาน
  โดยอัตโนมัติตามงานที่กำลังดำเนินการ
- **คำสั่ง** — คำสั่งแบบ slash ที่ผู้ใช้เรียกใช้โดยตรง
  (เช่น `/review-mdx`)
- **เอเจนต์** — ซับเอเจนต์เฉพาะทางที่จัดการงานซับซ้อน
  หลายขั้นตอนโดยอัตโนมัติ
