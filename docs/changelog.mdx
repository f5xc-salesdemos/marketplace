Notable changes to the f5xc-salesdemos marketplace.

## 1.0.0

Initial release of the marketplace with two plugins.

### f5xc-docs-tools v1.0.0

- MDX content validation skill with seven checks
  (frontmatter, syntax, imports, components, images,
  structure, exports)
- `/review-mdx` command for on-demand validation

### f5xc-sales-engineer v1.0.0

- Four persona skills: sales-engineer (dispatcher),
  demo-executor (4-stage lifecycle), presenter
  (walkthrough), subject-matter-expert (cited Q&A)
- Two agents: demo-housekeeping (readiness verification
  and teardown), demo-researcher (reference-backed
  research)
- Convention file architecture for product-agnostic
  reuse across demo repositories
