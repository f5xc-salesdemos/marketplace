Das Plugin **f5xc-platform** bietet vollständige Automatisierung für die
F5 Distributed Cloud Plattform. Es umfasst Web-Konsolenoperationen über
Chrome DevTools MCP, spezifikationsbewusste REST-API-Operationen über cURL,
Multi-Provider-Authentifizierung und Konfigurationsanalyse über
98 Ressourcentypen in 38 Domänen.

<Badge text="v3.1.1" variant="note" /> <Badge text="Produktivität" variant="tip" />

## Installation

```
/plugin install f5xc-platform@f5-sales-demo-marketplace
```

## Fähigkeiten

### api-operations

Spezifikationsbewusste API-CRUD-Operationen, die 98 Ressourcentypen über
38 Domänen abdecken. Bietet präzise Endpunkte, Payload-Vorlagen,
Abhängigkeitsreihenfolgen und mehrstufige Workflow-Kompositionen für
HTTP-Load-Balancer, Origin-Pools, WAF-Richtlinien, DNS-Zonen,
Zertifikate, Cloud-Sites und mehr.

### config-analysis

Konfigurationsanalyse- und Beratungsfähigkeit. Analysiert JSON-Konfigurationen
von Kunden, um Fragen zu Sicherheitsstatus, Feature-Aktivierung,
Moduseinstellungen, Ausnahmen und Best Practices zu beantworten.
Unterstützt mehrstufige Frage-und-Antwort-Sitzungen mit komprimiertem
Kontext zwischen den Weiterleitungen.

### api-auth

API-Authentifizierung mit Unterstützung für API-Token-Header-Authentifizierung
und P12-Zertifikat-Authentifizierung mit Token-Validierungsverfahren.

### console-auth

Multi-Provider-Authentifizierung für die F5 XC Web-Konsole.
Erkennt automatisch den Anmeldetyp: native F5 XC-Anmeldung, Azure SSO mit
zwischengespeicherter Sitzung oder Azure SSO mit DUO Verified Push MFA.

### console-navigator

Deterministische Navigation zu Bereichen der F5 XC-Konsole. Übersetzt
Bereichsnamen in URL-Pfade, überprüft den Sitzungszustand und bestätigt
das Laden der Seite.

### platform-index / api-index / console-index

Intent-Router, die Benutzeranfragen klassifizieren und an die
entsprechende Fähigkeit und den zuständigen Agenten weiterleiten. Diese
werden automatisch aktiviert und sind nicht vom Benutzer aufrufbar.

## Befehle

### /check-api-token

```
/f5xc-platform:check-api-token
```

Validiert Ihr F5 XC API-Token durch Delegation an den api-operator-Agenten.

### /login-console

```
/f5xc-platform:login-console
```

Authentifiziert sich bei der F5 Distributed Cloud Konsole über den
console-operator-Agenten.

## Agenten

### api-operator

Autonomer REST-API-Agent, der cURL- und jq-Sequenzen für
Ressourcen-CRUD, Token-Validierung und Konfigurationsoperationen ausführt.
Läuft in einem Subagenten, um umfangreiche API-JSON-Payloads aus dem
Hauptsitzungskontext fernzuhalten.

### console-operator

Autonomer Browser-Automatisierungsagent, der Chrome DevTools MCP
Tool-Sequenzen für Authentifizierung, Navigation und Formularinteraktionen
ausführt. Läuft in einem Subagenten, um token-intensive Browser-Snapshots
aus der Hauptsitzung fernzuhalten.

### config-analyzer

Schreibgeschützter Konfigurationsanalyse-Agent. Analysiert JSON-Konfigurationen
von Kunden anhand von Ressourcenprofil-Referenzdateien, um Fragen zu
Sicherheitsstatus, Feature-Aktivierung, Modusübergängen und Best Practices
zu beantworten.

<Aside type="caution">
  Der config-analyzer-Agent ist **kurzlebig** &mdash; er wird für
  jede Frage neu erzeugt und hat kein Gedächtnis zwischen den
  Weiterleitungen. Folgefragen müssen komprimierte vorherige Ergebnisse
  im Weiterleitungs-Prompt enthalten.
</Aside>
