Das **f5xc-meddpicc**-Plugin bietet ein umfassendes MEDDPICC-Framework
für Vertriebsqualifizierung und Deal-Execution bei komplexen
B2B-Unternehmensverkäufen. Es verwandelt MEDDPICC von einer reinen
Checklisten-Übung in ein lebendiges Deal-Execution-System mit
evidenzbasierter Qualifizierung, strukturierten Deal-Reviews und
umsetzbarem Coaching.

<Badge text="v1.0.4" variant="note" /> <Badge text="Produktivität" variant="tip" />

## Installation

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## Was ist MEDDPICC?

MEDDPICC ist ein Qualifizierungs- und Deal-Execution-Framework,
das für komplexe B2B-Verkäufe entwickelt wurde, bei denen mehrere
Stakeholder, technische Risiken und Beschaffungsschritte den
Fortschritt gefährden können.

| Buchstabe | Element | Zweck |
| --------- | ------- | ----- |
| **M** | Metrics | Quantifizierte Geschäftsergebnisse, die der Kunde erwartet |
| **E** | Economic Buyer | Person mit finaler Budgetautorität |
| **D** | Decision Criteria | Anforderungen zur Bewertung der Optionen |
| **D** | Decision Process | Schrittweiser Weg von der Evaluierung bis zur Vereinbarung |
| **P** | Paper Process | Beschaffungsmechanismen (Recht, Sicherheit, Unterschrift) |
| **I** | Identify Pain | Überzeugende geschäftliche Schmerzpunkte, die Dringlichkeit erzeugen |
| **C** | Champion | Einflussreicher interner Fürsprecher |
| **C** | Competition | Alle Alternativen (Anbieter, Eigenentwicklung, Nichtstun) |

## Skills

### meddpicc-coach

Umfassendes MEDDPICC-Methodik-Coaching, das sich automatisch
aktiviert, wenn Sie Deal-Qualifizierung, Pipeline-Gesundheit,
Forecast-Genauigkeit oder ein beliebiges MEDDPICC-Element besprechen.
Bietet evidenzbasierte Anleitung, Empfehlungen für den operativen
Rhythmus und Links zu detailliertem Referenzmaterial.

<Aside type="tip" title="Automatisch aktiviert">
  Dieser Skill aktiviert sich automatisch bei Relevanz — kein
  Slash-Befehl erforderlich.
</Aside>

### deal-qualification

Erstellt eine bewertete MEDDPICC-Scorecard (0&ndash;24) mit
Lückenanalyse, Risikobewertung und Forecast-Empfehlung. Jedes
Element wird auf Basis verifizierter Evidenz mit 0&ndash;3 bewertet.

### deal-review

Ermöglicht strukturierte wöchentliche Deal-Inspektionen mit
Evidenz-Delta-Tracking, Risikoidentifikation und Maßnahmen mit
benannten Verantwortlichen und Fälligkeitsdaten.

### mutual-action-plan

Erstellt einen kundenorientierten Mutual Action Plan, der an
MEDDPICC-Elemente gekoppelt ist. Umfasst Evaluierungsmeilensteine,
Beschaffungs-Roadmap, Implementierungsbereitschaft und
Stakeholder-Mapping.

### champion-test

Wendet eine 5-Kriterien-Bewertung (Power, Personal Win, Access,
Intel, Action) mit einer Punktzahl von 0&ndash;15 an, um
festzustellen, ob Ihr interner Kontakt ein echter Champion,
aufstrebender Champion, Coach oder freundlicher Kontakt ist.

## Befehle

| Befehl | Zweck |
| ------ | ----- |
| `/f5xc-meddpicc:qualify-deal [account]` | Deal mit MEDDPICC-Scorecard bewerten |
| `/f5xc-meddpicc:deal-review [account]` | Wöchentliche Deal-Inspektion durchführen |
| `/f5xc-meddpicc:champion-test [contact]` | Prüfen, ob ein Kontakt ein echter Champion ist |
| `/f5xc-meddpicc:build-map [account]` | Mutual Action Plan erstellen |

## Agenten

### deal-analyst

Schreibgeschützter Research-Agent, der die Deal-Gesundheit durch
Analyse lokaler Dokumentation, Meeting-Notizen und Account-Pläne
bewertet. Erstellt strukturierte MEDDPICC-Bewertungsberichte mit
Evidenzzitaten, Lückenanalyse und priorisierten
Handlungsempfehlungen.

## Referenzbibliothek

Das Plugin enthält umfassendes Referenzmaterial, das bei Bedarf
geladen wird:

- **MEDDPICC-Framework** &mdash; vollständige Definitionen und
  Evidenzanforderungen für alle 8 Elemente
- **Discovery-Fragen** &mdash; praxiserprobte Fragen für jedes
  Element über den gesamten Deal-Zyklus
- **Rollen &amp; Verantwortlichkeiten** &mdash; RACI-Zuständigkeitsmatrix
  nach Teamrolle (AE, SE, CSM, PS, Product, Channel, Deal Desk, Legal)
- **Deal-Stage-Gates** &mdash; was erfüllt sein muss, bevor der
  nächste Schritt erfolgt (von Early Qualification bis Closed Won)
- **Anti-Patterns** &mdash; 10 häufige Fehlermuster mit
  evidenzbasierter Behebung
- **Scoring-Rubrik** &mdash; objektive 0&ndash;3-Bewertungskriterien
  pro Element
- **Review-Vorlagen** &mdash; wöchentliche Inspektionsfragen-Sets mit
  Red-Flag-Indikatoren
- **MAP-Vorlagen** &mdash; kundenorientierte Aktionsplan-Struktur
  mit Evaluierungs-, Beschaffungs- und Implementierungs-Roadmaps
