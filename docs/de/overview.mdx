Ein **Claude Code Plugin-Marktplatz** ist eine kuratierte Sammlung von
Plugins, die Claude Code um neue Fähigkeiten, Befehle und
Agenten erweitern. Marktplätze ermöglichen es Teams, Werkzeuge zu teilen,
sodass jeder Entwickler die gleichen Funktionen erhält, ohne manuelle Einrichtung.

Der **f5xc-salesdemos Marktplatz** bietet Plugins, die für die
[f5xc-salesdemos](https://github.com/f5xc-salesdemos)
Dokumentationsplattform entwickelt wurden — ein Ökosystem aus über 20 Repositories,
die Astro/Starlight-Websites über GitHub Pages mit einer gemeinsamen
Build-Pipeline veröffentlichen.

## Funktionsweise

1. **Fügen Sie den Marktplatz** zu Claude Code hinzu — dadurch wird er
   als Plugin-Quelle registriert
2. **Installieren Sie Plugins** vom Marktplatz — jedes Plugin fügt
   Ihrer Claude Code-Sitzung Fähigkeiten und Befehle hinzu
3. **Nutzen Sie die Werkzeuge** — Plugins werden automatisch je nach
   Kontext aktiviert, oder Sie rufen ihre Befehle direkt auf

## Verfügbare Plugins

| Plugin | Kategorie | Beschreibung |
|--------|-----------|--------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | Produktivität | MDX-Inhaltsvalidierung und Review-Werkzeuge für Dokumentations-Repositories |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | Produktivität | Sales Engineer Persona-Framework — Demo-Durchführung, Präsentation, Q&A, Umgebungsmanagement |

## Schnellstart

Fügen Sie den Marktplatz hinzu und installieren Sie das Docs-Tools-Plugin:

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

Siehe [Erste Schritte](/marketplace/getting-started/) für eine
vollständige Anleitung.

## Architektur

Ein Marktplatz ist ein Git-Repository, das ein oder mehrere
Plugins enthält. Jedes Plugin bündelt Fähigkeiten, Befehle und Agenten
in einer einzelnen installierbaren Einheit.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marktplatz** — die Registry. Listet verfügbare Plugins
  mit Metadaten für Auffindbarkeit und Installation auf.
- **Plugin** — die installierbare Einheit. Enthält eine oder mehrere
  Fähigkeiten, Befehle oder Agenten.
- **Fähigkeit (Skill)** — kontextuelles Wissen, das Claude automatisch
  basierend auf der jeweiligen Aufgabe aktiviert.
- **Befehl (Command)** — ein Slash-Befehl, den der Benutzer direkt aufruft
  (z. B. `/review-mdx`).
- **Agent** — ein spezialisierter Unter-Agent, der komplexe,
  mehrstufige Aufgaben autonom bearbeitet.
