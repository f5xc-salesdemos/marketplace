La place de marché f5-sales-demo fournit les plugins suivants.
Chaque plugin ajoute des compétences, des commandes et des agents à Claude Code
qui facilitent la documentation, les flux de travail de démonstration, la méthodologie de vente,
l'image de marque et les opérations de développement.

## Que contient un plugin ?

Chaque plugin peut inclure :

- **Compétences** — connaissances contextuelles que Claude active
  automatiquement lorsque cela est pertinent (p. ex., révision de fichiers MDX)
- **Commandes** — commandes slash que vous invoquez directement
  (p. ex., `/f5xc-docs-tools:review-mdx`)
- **Agents** — sous-agents spécialisés pour les tâches complexes

## Installer un plugin

```
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
/plugin install f5xc-brand@f5-sales-demo-marketplace
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
/plugin install f5xc-platform@f5-sales-demo-marketplace
/plugin install osint-framework@f5-sales-demo-marketplace
/plugin install f5xc-firecrawl@f5-sales-demo-marketplace
```

guide de configuration complet.
