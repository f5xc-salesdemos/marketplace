Le marketplace f5xc-salesdemos fournit les plugins suivants.
Chaque plugin ajoute des compétences, des commandes et des agents à Claude Code
qui facilitent la documentation, les workflows de démonstration, la méthodologie de vente,
l'identité de marque et les opérations de développement.

## Que contient un Plugin ?

Chaque plugin peut inclure :

- **Compétences** — connaissances contextuelles que Claude active
  automatiquement lorsqu'elles sont pertinentes (par ex., revue de fichiers MDX)
- **Commandes** — commandes slash que vous invoquez directement
  (par ex., `/f5xc-docs-tools:review-mdx`)
- **Agents** — sous-agents spécialisés pour les tâches complexes

/plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
