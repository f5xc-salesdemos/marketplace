f5xc-salesdemos 마켓플레이스는 다음 플러그인을 제공합니다.
각 플러그인은 문서화, 데모 워크플로우, 영업 방법론,
브랜딩 및 개발 운영에 도움이 되는 스킬, 명령어, 에이전트를
Claude Code에 추가합니다.

|--------|---------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | 1.1.2 | 생산성 | MDX 콘텐츠 검증 및 리뷰 도구 |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | 1.0.6 | 생산성 | 세일즈 엔지니어 페르소나 프레임워크 — 데모 실행, 프레젠테이션, Q&A, 환경 관리 |

| [f5xc-docs-pipeline](/marketplace/plugins/f5xc-docs-pipeline/) | 1.0.2 | 생산성 | 문서화 파이프라인 아키텍처 — 콘텐츠 저작, 로컬 미리보기, 설정 관리 |

## 플러그인에 포함된 것은?

각 플러그인에는 다음이 포함될 수 있습니다:

- **스킬** — 관련 상황에서 Claude가 자동으로 활성화하는
  컨텍스트 지식 (예: MDX 파일 리뷰)
- **명령어** — 직접 호출하는 슬래시 명령어
  (예: `/f5xc-docs-tools:review-mdx`)
- **에이전트** — 복잡한 작업을 위한 전문 하위 에이전트
