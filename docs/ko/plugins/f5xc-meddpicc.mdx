**f5xc-meddpicc** 플러그인은 복잡한 엔터프라이즈 B2B 영업을 위한 포괄적인
MEDDPICC 영업 자격 평가 및 딜 실행 프레임워크를 제공합니다. MEDDPICC를
단순한 체크리스트 작업에서 증거 기반 자격 평가, 구조화된 딜 리뷰, 실행
가능한 코칭을 갖춘 살아있는 딜 실행 시스템으로 전환합니다.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## 설치

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## MEDDPICC란 무엇인가?

MEDDPICC는 다수의 이해관계자, 기술적 리스크, 조달 절차가 모멘텀을
저해할 수 있는 복잡한 B2B 영업을 위해 설계된 자격 평가 및 딜 실행
프레임워크입니다.

| 문자 | 요소 | 목적 |
| ------ | ------- | ------- |
| **M** | Metrics (지표) | 고객이 기대하는 정량화된 비즈니스 성과 |
| **E** | Economic Buyer (경제적 구매자) | 최종 예산 권한을 가진 개인 |
| **D** | Decision Criteria (의사결정 기준) | 옵션을 평가하는 데 사용되는 요구사항 |
| **D** | Decision Process (의사결정 프로세스) | 평가에서 합의까지의 단계별 경로 |
| **P** | Paper Process (문서 프로세스) | 조달 메커니즘 (법무, 보안, 서명) |
| **I** | Identify Pain (고통 식별) | 긴급성을 유발하는 강력한 비즈니스 고통 |
| **C** | Champion (챔피언) | 강력한 내부 옹호자 |
| **C** | Competition (경쟁) | 모든 대안 (벤더, 자체 구축, 현상 유지) |

## 스킬

### meddpicc-coach

딜 자격 평가, 파이프라인 건전성, 예측 정확도 또는 MEDDPICC 요소에 대해
논의할 때 자동으로 활성화되는 포괄적인 MEDDPICC 방법론 코칭입니다.
증거 기반 가이드, 운영 리듬 권장 사항 및 상세 참조 자료 링크를
제공합니다.

<Aside type="tip" title="자동 활성화">
  이 스킬은 관련 내용이 감지되면 자동으로 활성화됩니다 — 슬래시 명령어가
  필요하지 않습니다.
</Aside>

### deal-qualification

갭 분석, 리스크 평가 및 예측 권장 사항이 포함된 MEDDPICC 스코어카드
(0&ndash;24)를 생성합니다. 각 요소는 검증된 증거를 기반으로
0&ndash;3으로 점수가 매겨집니다.

### deal-review

증거 델타 추적, 리스크 식별, 담당자 및 기한이 지정된 액션 아이템을
포함하는 구조화된 주간 딜 점검을 수행합니다.

### mutual-action-plan

MEDDPICC 요소에 연결된 고객 대면 공동 액션 플랜(Mutual Action Plan)을
구축합니다. 평가 마일스톤, 조달 로드맵, 구현 준비 상태 및 이해관계자
매핑을 포함합니다.

### champion-test

5가지 기준 평가(Power, Personal Win, Access, Intel, Action)를 적용하여
0&ndash;15로 점수를 매기고, 내부 연락 담당자가 진정한 챔피언인지,
성장 중인 챔피언인지, 코치인지, 또는 우호적 연락처인지를 판별합니다.

## 명령어

| 명령어 | 목적 |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | MEDDPICC 스코어카드로 딜 점수 매기기 |
| `/f5xc-meddpicc:deal-review [account]` | 주간 딜 점검 수행 |
| `/f5xc-meddpicc:champion-test [contact]` | 연락 담당자가 진정한 챔피언인지 평가 |
| `/f5xc-meddpicc:build-map [account]` | 공동 액션 플랜 구축 |

## 에이전트

### deal-analyst

로컬 문서, 회의 노트 및 계정 계획을 검토하여 딜 건전성을 분석하는
읽기 전용 리서치 에이전트입니다. 증거 인용, 갭 분석 및 우선순위가
지정된 액션 권장 사항이 포함된 구조화된 MEDDPICC 평가 보고서를
생성합니다.

## 참조 라이브러리

이 플러그인은 필요 시 로드되는 포괄적인 참조 자료를 포함합니다:

- **MEDDPICC 프레임워크** &mdash; 8개 요소 전체에 대한 완전한 정의 및
  증거 요구사항
- **디스커버리 질문** &mdash; 딜 사이클 전반에 걸쳐 모든 요소에 대해
  현장에서 검증된 질문
- **역할 및 책임** &mdash; 팀 역할별(AE, SE, CSM, PS, Product,
  Channel, Deal Desk, Legal) RACI 소유권 매트릭스
- **딜 스테이지 게이트** &mdash; 다음 단계로 진행하기 전에 충족해야
  하는 조건 (초기 자격 평가부터 Closed Won까지)
- **안티 패턴** &mdash; 증거 기반 해결 방안이 포함된 10가지 일반적인
  실패 모드
- **스코어링 루브릭** &mdash; 요소별 객관적 0&ndash;3 점수 기준
- **리뷰 템플릿** &mdash; 위험 신호 지표가 포함된 주간 점검 질문 세트
- **MAP 템플릿** &mdash; 평가, 조달 및 구현 로드맵이 포함된 고객 대면
  액션 플랜 구조
