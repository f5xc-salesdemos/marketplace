**f5xc-platform** 플러그인은 F5 Distributed Cloud 플랫폼을 위한 완전한 자동화를 제공합니다. Chrome DevTools MCP를 통한 웹 콘솔 운영, cURL을 통한 스펙 인식 REST API 운영, 다중 공급자 인증, 그리고 38개 도메인의 98개 리소스 유형에 대한 구성 분석을 포함합니다.

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## 설치

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## 스킬

### api-operations

38개 도메인에 걸친 98개 리소스 유형을 다루는 스펙 인식 API CRUD 운영. HTTP 로드 밸런서, 오리진 풀, WAF 정책, DNS 존, 인증서, 클라우드 사이트 등에 대한 정확한 엔드포인트, 페이로드 템플릿, 종속성 순서 지정 및 다단계 워크플로우 구성을 제공합니다.

### config-analysis

구성 분석 및 자문 기능. 고객 JSON 구성을 분석하여 보안 태세, 기능 활성화, 모드 설정, 제외 항목 및 모범 사례에 대한 질문에 답변합니다. 디스패치 간 압축된 컨텍스트를 활용한 다중 턴 Q&A를 지원합니다.

### api-auth

API 토큰 헤더 인증 및 P12 인증서 인증과 토큰 유효성 검사 절차를 지원하는 API 인증.

### console-auth

F5 XC 웹 콘솔을 위한 다중 공급자 인증. 로그인 유형을 자동 감지합니다: 네이티브 F5 XC 로그인, 캐시된 세션을 사용하는 Azure SSO, 또는 DUO 확인 푸시 MFA를 사용하는 Azure SSO.

### console-navigator

F5 XC 콘솔 섹션으로의 결정적 탐색. 섹션 이름을 URL 경로로 변환하고, 세션 상태를 확인하며, 페이지 로드를 검증합니다.

### platform-index / api-index / console-index

사용자 요청을 분류하고 적절한 스킬 및 에이전트에 위임하는 의도 라우터. 자동으로 활성화되며 사용자가 직접 호출할 수 없습니다.

## 명령어

### /check-api-token

```
/f5xc-platform:check-api-token
```

api-operator 에이전트에 위임하여 F5 XC API 토큰의 유효성을 검사합니다.

### /login-console

```
/f5xc-platform:login-console
```

console-operator 에이전트를 통해 F5 Distributed Cloud 콘솔에 인증합니다.

## 에이전트

### api-operator

리소스 CRUD, 토큰 유효성 검사 및 구성 운영을 위한 cURL 및 jq 시퀀스를 실행하는 자율 REST API 에이전트. 장황한 API JSON 페이로드를 메인 세션 컨텍스트에서 분리하기 위해 서브에이전트로 실행됩니다.

### console-operator

인증, 탐색 및 폼 상호작용을 위한 Chrome DevTools MCP 도구 시퀀스를 실행하는 자율 브라우저 자동화 에이전트. 토큰이 많은 브라우저 스냅샷을 메인 세션에서 분리하기 위해 서브에이전트로 실행됩니다.

### config-analyzer

읽기 전용 구성 분석 에이전트. 리소스 프로필 참조 파일을 기준으로 고객 JSON 구성을 분석하여 보안 태세, 기능 활성화, 모드 전환 및 모범 사례에 대한 질문에 답변합니다.

<Aside type="caution">
  config-analyzer 에이전트는 **임시적**입니다 &mdash; 각 질문마다 새로 생성되며 디스패치 간 메모리를 유지하지 않습니다. 후속 질문에는 디스패치 프롬프트에 이전 분석 결과를 압축하여 포함해야 합니다.
</Aside>
