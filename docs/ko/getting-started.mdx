이 가이드는 f5xc-salesdemos 마켓플레이스를 Claude Code에 추가하고 첫 번째 플러그인을 설치하는 과정을 안내합니다.

## 사전 요구 사항

- Claude Code **v1.0.33** 이상 (플러그인 마켓플레이스 지원)
- [f5xc-salesdemos](https://github.com/f5xc-salesdemos)
  조직에 접근 권한이 있는 GitHub 계정

## 설치

<Steps>

1. **마켓플레이스 추가**

   Claude Code 세션에서 다음 명령을 실행합니다:

   ```
   /plugin marketplace add f5xc-salesdemos/marketplace
   ```

   이 명령은 마켓플레이스를 플러그인 소스로 등록합니다. 한 번만
   수행하면 됩니다.

2. **사용 가능한 플러그인 탐색**

   마켓플레이스에서 제공하는 항목을 확인합니다:

   ```
   /plugin marketplace list f5xc-salesdemos-marketplace
   ```

3. **플러그인 설치**

   필요한 플러그인을 설치합니다:

   ```
   /plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
   /plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
   ```

4. **플러그인 사용**

   f5xc-salesdemos 콘텐츠 리포지토리로 이동합니다.

   **문서 도구**의 경우, MDX 검토 명령을 실행합니다:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   **세일즈 엔지니어**의 경우, "prepare the demo", "run the demo",
   또는 "walk through the demo"와 같은 트리거 문구를 사용합니다
   — 플러그인이 자동으로 활성화됩니다.

</Steps>

## 플러그인 업데이트

설치된 플러그인을 최신 버전으로 업데이트하려면:

```
/plugin update f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin update f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

## 제거

플러그인을 제거합니다:

```
/plugin uninstall f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin uninstall f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

마켓플레이스를 완전히 제거합니다:

```
/plugin marketplace remove f5xc-salesdemos-marketplace
```

## 팀 구성

팀 전체를 위해 마켓플레이스를 사전 구성하려면, 프로젝트의
`.claude/settings.json`에 추가합니다:

```json
{
  "extraKnownMarketplaces": [
    "f5xc-salesdemos/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5xc-salesdemos-marketplace",
    "f5xc-sales-engineer@f5xc-salesdemos-marketplace"
  ]
}
```

<Aside type="tip" title="공유 설정">
  `.claude/settings.json`을 리포지토리에 커밋하면 모든 팀원이
  마켓플레이스와 플러그인을 자동으로 받게 됩니다 — 수동 설정이
  필요하지 않습니다.
</Aside>

이를 통해 모든 기여자가 리포지토리에서 첫 번째 Claude Code 세션부터
동일한 도구를 사용할 수 있습니다.
