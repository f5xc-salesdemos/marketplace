**سوق إضافات Claude Code** هو مجموعة منسقة من
الإضافات التي توسع قدرات Claude Code بمهارات وأوامر
ووكلاء جدد. تتيح الأسواق للفرق مشاركة الأدوات بحيث يحصل
كل مطور على نفس الإمكانيات دون إعداد يدوي.

يوفر **سوق f5xc-salesdemos** إضافات مبنية لمنصة
التوثيق [f5xc-salesdemos](https://github.com/f5xc-salesdemos)
— وهي منظومة تضم أكثر من 20 مستودعًا تنشر مواقع
Astro/Starlight على GitHub Pages باستخدام خط بناء مشترك.

## كيف يعمل

1. **أضف السوق** إلى Claude Code — يؤدي هذا إلى تسجيله
   كمصدر للإضافات
2. **ثبّت الإضافات** من السوق — تضيف كل إضافة
   مهارات وأوامر إلى جلسة Claude Code الخاصة بك
3. **استخدم الأدوات** — تُفعّل الإضافات تلقائيًا بناءً على
   السياق، أو يمكنك استدعاء أوامرها مباشرة

## الإضافات المتاحة

| الإضافة | الفئة | الوصف |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | إنتاجية | أدوات التحقق من محتوى MDX ومراجعته لمستودعات التوثيق |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | إنتاجية | إطار عمل شخصية مهندس المبيعات — تنفيذ العروض التوضيحية، العروض التقديمية، الأسئلة والأجوبة، إدارة البيئات |

## البدء السريع

أضف السوق وثبّت إضافة أدوات التوثيق:

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

راجع [البدء](/marketplace/getting-started/) للحصول على
شرح تفصيلي كامل.

## البنية المعمارية

السوق هو مستودع Git يحتوي على إضافة واحدة أو أكثر.
تُجمّع كل إضافة المهارات والأوامر والوكلاء في وحدة
قابلة للتثبيت واحدة.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **السوق** — السجل. يسرد الإضافات المتاحة
  مع البيانات الوصفية للاكتشاف والتثبيت.
- **الإضافة** — الوحدة القابلة للتثبيت. تحتوي على مهارة
  واحدة أو أكثر، أو أوامر، أو وكلاء.
- **المهارة** — معرفة سياقية يُفعّلها Claude
  تلقائيًا بناءً على المهمة المطلوبة.
- **الأمر** — أمر مائل يستدعيه المستخدم مباشرة
  (مثل `/review-mdx`).
- **الوكيل** — وكيل فرعي متخصص يتعامل مع المهام
  المعقدة متعددة الخطوات بشكل مستقل.
