يوفر إضافة **osint-framework** كتالوج أدوات الاستخبارات مفتوحة المصدر
ومهارات التحقيق — 1,064 أداة مجانية لجمع المعلومات الاستخباراتية
عبر 34 فئة مُعيّنة من [osintframework.com](https://osintframework.com).
يتضمن مهارات قائمة على الفئات، وخطوط أنابيب تحقيق قابلة للتنفيذ،
وتنفيذ أدوات سطر الأوامر، وسير عمل واعٍ بأمن العمليات (OPSEC)،
ورسم بياني كيانات دائم للربط المتبادل عبر التحقيقات.

<Badge text="v1.0.1" variant="note" /> <Badge text="الأمان" variant="tip" />

## التثبيت

```
/plugin install osint-framework@f5xc-salesdemos-marketplace
```

## الأوامر

### /osint-investigate

تشغيل تحقيق OSINT كامل ضد أي هدف. يكتشف تلقائياً
نوع الهدف (بريد إلكتروني، نطاق، عنوان IP، اسم مستخدم، شركة،
اسم شخص) ويوجّه إلى مهارات الفئة الصحيحة.

```
/osint-investigate j.smith@example.com
/osint-investigate example.com
/osint-investigate 198.51.100.1
/osint-investigate example-user
```

### /osint-search

البحث في كتالوج الأدوات عن أدوات تطابق استعلام قدرة معينة.

```
/osint-search subdomain enumeration
/osint-search email breach check
/osint-search satellite imagery
```

### /osint-catalog

تصفح كتالوج الأدوات الكامل حسب الفئة. بدون وسائط يعرض
جميع الفئات الـ 34 مع عدد الأدوات.

```
/osint-catalog
/osint-catalog domain
/osint-catalog threat-intel
```

## المهارات

### osint-index

موجّه الأهداف على المستوى الأعلى. يُفعَّل تلقائياً عندما تقول "ابحث عن
معلومات حول"، "حقق"، "ابحث عن"، "فحص خلفية"،
أو تصف أي مهمة استخبارات مفتوحة المصدر. يوجّه إلى
مهارة الفئة الصحيحة بناءً على نوع الهدف.

| الهدف | يُوجَّه إلى |
|--------|----------|
| اسم مستخدم أو معرّف | `username-recon` |
| عنوان بريد إلكتروني | `email-recon` |
| اسم نطاق | `domain-recon` |
| عنوان IP | `ip-address-recon` |
| اسم شخص + شركة | `people-search` + `business-records` |
| شركة / مؤسسة | `business-records` + `domain-recon` |
| بصمة برنامج ضار | `malicious-file-analysis` + `threat-intelligence` |

### osint-catalog

متصفح كتالوج الأدوات. يوفر الوصول إلى سجل الأدوات الكامل،
وفهرس الفئات، وخطوط أنابيب التحقيق، ومرجع حدود المعدل،
ووثائق محرك الربط.

## الوكلاء

### osint-investigator

مُنسّق تحقيقات متعدد الأدوات مستقل. يُشغّل سير عمل
التحقيقات الكاملة — فحص توفر الأدوات، واكتشاف نوع الهدف،
وتنفيذ الأدوات بشكل تسلسلي (السلبية أولاً)، وتجميع النتائج،
وإنشاء تقارير منظمة بمستويات الثقة وتصنيفات أمن العمليات.

يتكامل مع رسم الكيانات البياني: كل نتيجة أداة تُنشئ كيانات
وعلاقات في `/tmp/osint-graph/` للربط المتبادل عبر التحقيقات.

<Aside type="caution">
  أدوات الاستطلاع النشط (nmap, nuclei, sherlock) تتصل مباشرةً
  بالبنية التحتية للهدف. تحذير أمن العمليات يُنبّه قبل
  تشغيل الأدوات النشطة. تأكد دائماً من حصولك على التصريح.
</Aside>

### osint-researcher

وكيل بحث في الكتالوج للقراءة فقط. يبحث في ملفات المراجع
لتوصية أفضل الأدوات لمهمة معينة. لا يُنفّذ أدوات أبداً —
يوصي ويشرح فقط. يُستخدم داخلياً بواسطة مهارات الفئات
لاكتشاف الأدوات.

### osint-executor

وكيل تنفيذ أداة واحدة عبر سطر الأوامر. يُشغّل أداة واحدة في كل مرة مع
معلمات محددة، ويُحلّل المخرجات، ويُعيد نتائج منظمة.
يتضمن وصفات تنفيذ لكل أداة لأكثر من 20 أداة مع العلامات الدقيقة،
وتنسيقات المخرجات، وأنماط تحليل jq.

## تحديد المعدل

جميع استدعاءات API تستخدم المساعد `osint_curl` مع تراجع أسي
مدمج عند HTTP 429/503. الحدود الرئيسية:

| واجهة API | الحد المجاني | متغير البيئة |
|-----|-----------|---------|
| ipinfo.io | 1,000/يوم | — |
| crt.sh | 60/دقيقة | — |
| NVD | 5/30 ثانية (بدون مفتاح) | `NVD_API_KEY` |
| OpenCorporates | يتطلب مفتاح | `OPENCORPORATES_API_KEY` |
| Shodan | 1/ثانية | `SHODAN_API_KEY` |
| GitHub API | 60/ساعة (بدون مصادقة) | `GH_TOKEN` |

واجهات API التي تتطلب مفاتيح يتم تخطيها تلقائياً مع رسالة
واضحة إذا لم يتم تعيين متغير البيئة.

## محرك الربط

يتضمن الإضافة رسماً بيانياً دائماً لكيانات JSON في
`/tmp/osint-graph/` يُجمّع النتائج عبر التحقيقات.

```bash
source plugins/osint-framework/scripts/osint-graph.sh
osint_graph_init

# Add entities
P=$(osint_entity_add "person" "J. Smith" --tool github-api)
C=$(osint_entity_add "company" "F5" --tool github-api)

# Link them
osint_rel_add "$P" "$C" "works_at" --tool github-api

# Query: all entities within 2 hops of the person
osint_graph_query "$P" 2

# Generate report
osint_graph_report
```

**تسجيل الثقة** يستخدم دمج متعدد المصادر:
`confidence = 1 - product(1 - ci)` لكل مصدر مؤكد.
مصدران بثقة 0.80 → ثقة مجمعة 0.96.

## إشعار قانوني

جميع الأدوات تستخدم فقط المعلومات المتاحة للعموم. يتحمل المستخدمون
مسؤولية ضمان الامتثال للقوانين واللوائح المعمول بها
وشروط خدمة المنصات. لا تستخدم أدوات OSINT أبداً
للوصول غير المصرح به أو التحرش أو أي نشاط غير قانوني.
