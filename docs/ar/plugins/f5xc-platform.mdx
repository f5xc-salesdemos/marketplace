يوفر المكون الإضافي **f5xc-platform** أتمتة كاملة لمنصة F5
Distributed Cloud. يغطي عمليات وحدة التحكم عبر الويب من خلال
Chrome DevTools MCP، وعمليات REST API المدركة للمواصفات عبر cURL،
والمصادقة متعددة المزودين، وتحليل التكوين عبر
98 نوع موارد في 38 مجالاً.

<Badge text="v3.1.1" variant="note" /> <Badge text="الإنتاجية" variant="tip" />

## التثبيت

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## المهارات

### api-operations

عمليات CRUD للواجهة البرمجية المدركة للمواصفات تغطي 98 نوع موارد عبر
38 مجالاً. توفر نقاط نهاية دقيقة، وقوالب حمولات البيانات،
وترتيب التبعيات، وتركيبات سير العمل متعددة الخطوات لموازنات
تحميل HTTP، ومجمعات الأصل، وسياسات WAF، ومناطق DNS،
والشهادات، والمواقع السحابية، والمزيد.

### config-analysis

قدرة تحليل التكوين والتوصيات. يحلل تكوينات JSON الخاصة بالعملاء
للإجابة عن أسئلة حول الوضع الأمني، وتفعيل الميزات، وإعدادات
الأوضاع، والاستثناءات، وأفضل الممارسات. يدعم الأسئلة والأجوبة
متعددة الجولات مع سياق مضغوط بين عمليات الإرسال.

### api-auth

مصادقة واجهة API تدعم مصادقة رأس رمز API ومصادقة شهادة P12
مع إجراءات التحقق من صحة الرمز.

### console-auth

مصادقة متعددة المزودين لوحدة تحكم F5 XC عبر الويب.
تكتشف تلقائياً نوع تسجيل الدخول: تسجيل دخول F5 XC الأصلي،
أو تسجيل الدخول الموحد عبر Azure مع جلسة مخزنة مؤقتاً،
أو تسجيل الدخول الموحد عبر Azure مع المصادقة متعددة العوامل
عبر DUO بالدفع المُتحقق منه.

### console-navigator

تنقل حتمي إلى أقسام وحدة تحكم F5 XC. يترجم أسماء الأقسام
إلى مسارات URL، ويتحقق من صحة الجلسة، ويؤكد تحميل الصفحة.

### platform-index / api-index / console-index

موجهات النوايا التي تصنف طلبات المستخدم وتفوضها إلى المهارة
والوكيل المناسبين. تعمل هذه تلقائياً وليست قابلة للاستدعاء
من قبل المستخدم.

## الأوامر

### /check-api-token

```
/f5xc-platform:check-api-token
```

يتحقق من صحة رمز F5 XC API الخاص بك عن طريق التفويض إلى وكيل
api-operator.

### /login-console

```
/f5xc-platform:login-console
```

يُجري المصادقة على وحدة تحكم F5 Distributed Cloud عبر وكيل
console-operator.

## الوكلاء

### api-operator

وكيل REST API مستقل ينفذ تسلسلات cURL وjq لعمليات CRUD
على الموارد، والتحقق من الرموز، وعمليات التكوين. يعمل كوكيل
فرعي للحفاظ على حمولات JSON المطولة للواجهة البرمجية خارج
سياق الجلسة الرئيسية.

### console-operator

وكيل أتمتة متصفح مستقل ينفذ تسلسلات أدوات Chrome DevTools MCP
للمصادقة والتنقل والتفاعل مع النماذج. يعمل كوكيل فرعي
للحفاظ على لقطات المتصفح الثقيلة بالرموز خارج الجلسة الرئيسية.

### config-analyzer

وكيل تحليل تكوين للقراءة فقط. يحلل تكوينات JSON الخاصة بالعملاء
مقابل ملفات مرجع ملفات تعريف الموارد للإجابة عن أسئلة حول
الوضع الأمني، وتفعيل الميزات، وانتقالات الأوضاع،
وأفضل الممارسات.

<Aside type="caution">
  وكيل config-analyzer **مؤقت** &mdash; يتم إنشاؤه من جديد
  لكل سؤال بدون ذاكرة بين عمليات الإرسال. يجب أن تتضمن
  أسئلة المتابعة النتائج السابقة المضغوطة في موجه الإرسال.
</Aside>
