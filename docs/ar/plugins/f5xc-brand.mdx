يفرض إضافة **f5xc-brand** هوية العلامة التجارية لشركة F5
عبر جميع أنواع المحتوى. يغطي لوحة ألوان F5 المكونة من 45 لونًا،
وقواعد الطباعة (عناوين Neusa Next Pro Wide، ونص Proxima Nova
الأساسي)، واستخدام الشعارات بصيغة SVG فقط، واختيار حزم الأيقونات،
وإمكانية الوصول وفق معايير WCAG AA، ومصطلحات المنتجات.

<Badge text="v1.0.2" variant="note" /> <Badge text="الإنتاجية" variant="tip" />

## التثبيت

```
/plugin install f5xc-brand@f5-sales-demo-marketplace
```

## المهارات

### brand-guardian

تطبيق استباقي لهوية العلامة التجارية عند إنشاء أي محتوى.
يتم تفعيله تلقائيًا عند إنشاء الوثائق، والمخططات،
والعروض التقديمية، والصور المصغرة للفيديو، ورسومات وسائل التواصل الاجتماعي،
ومخططات Mermaid، أو رسومات Excalidraw. يضمن الاستخدام الصحيح لألوان F5،
والطباعة، والشعارات، والأيقونات، ونبرة الصوت، وإمكانية الوصول.

### brand-reviewer

مراجعة المحتوى الحالي للتحقق من امتثاله لمعايير العلامة التجارية لـ F5. يقوم بتدقيق الألوان،
والطباعة، وصيغة الشعار، واختيار الأيقونات، والنص البديل، ونسب التباين،
ومصطلحات المنتجات. ينتج تقارير منظمة تتضمن
المخالفات، والتحذيرات، والاقتراحات.

### visual-content

قواعد خاصة بكل صيغة للمحتوى المرئي ذي العلامة التجارية. يوفر
إرشادات لمخططات Mermaid، ورسومات Excalidraw، ولقطات الشاشة،
وعروض PowerPoint التقديمية، والصور المصغرة لـ YouTube، ونصوص الفيديو،
ورسومات وسائل التواصل الاجتماعي مع اختيار حزم الأيقونات، وتخطيطات الشرائح،
والخطوط البديلة، والأبعاد الخاصة بكل منصة.

## الأوامر

### /review-brand

```
/f5xc-brand:review-brand [path-or-glob]
```

يراجع المحتوى للتحقق من امتثاله لمعايير العلامة التجارية لـ F5. يقبل مسارًا اختياريًا
أو نمط glob لتحديد نطاق المراجعة. بدون وسائط، يراجع
جميع المحتوى في المجلد الحالي.
