يوفر إضافة **f5xc-meddpicc** إطار عمل شامل لتأهيل المبيعات وتنفيذ
الصفقات وفق منهجية MEDDPICC مصمم لمبيعات المؤسسات المعقدة بين الشركات
(B2B). يحوّل MEDDPICC من مجرد تمرين لتحديد مربعات الاختيار إلى نظام
حي لتنفيذ الصفقات مع تأهيل مبني على الأدلة، ومراجعات منظمة للصفقات،
وتوجيه عملي قابل للتنفيذ.

```
/plugin install f5xc-meddpicc@f5-sales-demo-marketplace
```

## ما هو MEDDPICC؟

MEDDPICC هو إطار عمل للتأهيل وتنفيذ الصفقات مصمم لمبيعات B2B المعقدة
حيث يمكن لتعدد أصحاب المصلحة والمخاطر التقنية وخطوات المشتريات أن
تعرقل الزخم.

| الحرف | العنصر | الغرض |
| ------ | ------- | ------- |
| **M** | المقاييس (Metrics) | نتائج الأعمال المحددة كمياً التي يتوقعها العميل |
| **E** | المشتري الاقتصادي (Economic Buyer) | الشخص الذي يملك صلاحية الموافقة النهائية على الميزانية |
| **D** | معايير القرار (Decision Criteria) | المتطلبات المستخدمة لتقييم الخيارات |
| **D** | عملية القرار (Decision Process) | المسار خطوة بخطوة من التقييم إلى الاتفاق |
| **P** | العملية الورقية (Paper Process) | آليات المشتريات (القانونية، الأمنية، التوقيع) |
| **I** | تحديد الألم (Identify Pain) | ألم تجاري مقنع يخلق شعوراً بالإلحاح |
| **C** | البطل (Champion) | مناصر داخلي مؤثر |
| **C** | المنافسة (Competition) | جميع البدائل (موردون، بناء ذاتي، عدم فعل شيء) |

## المهارات

### meddpicc-coach

تدريب شامل على منهجية MEDDPICC يتم تفعيله تلقائياً عند مناقشة تأهيل
الصفقات أو صحة خط الأنابيب أو دقة التوقعات أو أي عنصر من عناصر
MEDDPICC. يوفر توجيهاً مبنياً على الأدلة، وتوصيات لإيقاع العمل،
وروابط لمواد مرجعية مفصلة.

<Aside type="tip" title="تفعيل تلقائي">
  يتم تفعيل هذه المهارة تلقائياً عند الحاجة — لا حاجة لأمر مائل.
</Aside>

### deal-qualification

ينتج بطاقة تقييم MEDDPICC مسجلة (0&ndash;24) مع تحليل الفجوات
وتقييم المخاطر وتوصية التوقعات. يتم تقييم كل عنصر من 0&ndash;3
بناءً على أدلة موثقة.

### deal-review

يسهّل عمليات فحص الصفقات الأسبوعية المنظمة مع تتبع فروق الأدلة
وتحديد المخاطر وبنود العمل مع مسؤولين محددين ومواعيد نهائية.

### mutual-action-plan

يبني خطة عمل متبادلة موجهة للعميل مرتبطة بعناصر MEDDPICC. تتضمن
معالم التقييم وخارطة طريق المشتريات والجاهزية للتنفيذ ورسم خرائط
أصحاب المصلحة.

### champion-test

يطبق تقييماً من 5 معايير (القوة، المكسب الشخصي، الوصول، المعلومات
الاستخباراتية، الفعل) بتقييم من 0&ndash;15 لتحديد ما إذا كان جهة
الاتصال الداخلية لديك بطلاً حقيقياً أو بطلاً ناشئاً أو مدرباً أو
جهة اتصال ودية.

## الوكلاء

### deal-analyst

وكيل بحث للقراءة فقط يحلل صحة الصفقة من خلال مراجعة الوثائق المحلية
وملاحظات الاجتماعات وخطط الحسابات. ينتج تقارير تقييم MEDDPICC منظمة
مع استشهادات بالأدلة وتحليل الفجوات وتوصيات إجراءات مرتبة حسب
الأولوية.

## المكتبة المرجعية

تتضمن الإضافة مواد مرجعية شاملة يتم تحميلها عند الطلب:

- **إطار عمل MEDDPICC** &mdash; تعريفات كاملة ومتطلبات الأدلة لجميع
  العناصر الثمانية
- **أسئلة الاستكشاف** &mdash; أسئلة مجربة ميدانياً لكل عنصر عبر
  دورة الصفقة
- **الأدوار والمسؤوليات** &mdash; مصفوفة ملكية RACI حسب دور الفريق
  (AE، SE، CSM، PS، المنتج، القناة، مكتب الصفقات، القانوني)
- **بوابات مراحل الصفقة** &mdash; ما يجب أن يكون صحيحاً قبل التقدم
  (من التأهيل المبكر إلى الإغلاق الناجح)
- **الأنماط المضادة** &mdash; 10 أنماط فشل شائعة مع معالجة مبنية
  على الأدلة
- **معايير التقييم** &mdash; معايير تقييم موضوعية من 0&ndash;3 لكل
  عنصر
- **قوالب المراجعة** &mdash; مجموعات أسئلة الفحص الأسبوعي مع
  مؤشرات الإنذار الأحمر
- **قوالب خطة العمل المتبادلة** &mdash; هيكل خطة عمل موجهة للعميل
  مع خرائط طريق التقييم والمشتريات والتنفيذ
