Il marketplace f5-sales-demo fornisce i seguenti plugin.
Ogni plugin aggiunge competenze, comandi e agenti a Claude Code
che aiutano con la documentazione, i flussi di lavoro demo, la metodologia di vendita,
il branding e le operazioni di sviluppo.

## Cosa contiene un Plugin?

Ogni plugin può includere:

- **Competenze** — conoscenze contestuali che Claude attiva
  automaticamente quando pertinenti (ad es., revisione di file MDX)
- **Comandi** — comandi slash che si invocano direttamente
  (ad es., `/f5xc-docs-tools:review-mdx`)
- **Agenti** — sotto-agenti specializzati per attività complesse

## Installazione di un Plugin

```
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
/plugin install f5xc-sales-engineer@f5-sales-demo-marketplace
/plugin install f5xc-github-ops@f5-sales-demo-marketplace
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace

/plugin install f5xc-devcontainer@f5-sales-demo-marketplace
/plugin install f5xc-platform@f5-sales-demo-marketplace
/plugin install osint-framework@f5-sales-demo-marketplace
/plugin install f5xc-firecrawl@f5-sales-demo-marketplace
```
