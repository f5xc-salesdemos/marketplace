Un **marketplace di plugin di Claude Code** è una raccolta curata di
plugin che estendono Claude Code con nuove competenze, comandi e
agenti. I marketplace permettono ai team di condividere gli strumenti
in modo che ogni sviluppatore ottenga le stesse funzionalità senza
configurazione manuale.

Il **Marketplace f5-sales-demo** fornisce plugin creati per
la piattaforma di documentazione
[f5-sales-demo](https://github.com/f5-sales-demo) — un
ecosistema di oltre 20 repository che pubblicano siti
Astro/Starlight su GitHub Pages utilizzando una pipeline di
build condivisa.

## Come Funziona

1. **Aggiungi il marketplace** a Claude Code — questo lo registra
   come sorgente di plugin
2. **Installa i plugin** dal marketplace — ogni plugin aggiunge
   competenze e comandi alla tua sessione di Claude Code
3. **Usa gli strumenti** — i plugin si attivano automaticamente
   in base al contesto, oppure puoi invocare i loro comandi
   direttamente

## Plugin Disponibili

| Plugin | Categoria | Descrizione |
|--------|-----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | Produttività | Strumenti di validazione e revisione di contenuti MDX per repository di documentazione |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | Produttività | Framework persona Sales Engineer — esecuzione demo, presentazioni, Q&A, gestione ambienti |

## Avvio Rapido

Aggiungi il marketplace e installa il plugin docs tools:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

Consulta [Per Iniziare](/marketplace/getting-started/) per una
guida completa passo dopo passo.

## Architettura

Un marketplace è un repository Git che contiene uno o più
plugin. Ogni plugin racchiude competenze, comandi e agenti
in un'unica unità installabile.

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **Marketplace** — il registro. Elenca i plugin disponibili
  con metadati per la scoperta e l'installazione.
- **Plugin** — l'unità installabile. Contiene una o più
  competenze, comandi o agenti.
- **Skill** — conoscenza contestuale che Claude attiva
  automaticamente in base al compito in corso.
- **Comando** — un comando slash che l'utente invoca
  direttamente (es., `/review-mdx`).
- **Agente** — un sotto-agente specializzato che gestisce
  compiti complessi e multi-fase in modo autonomo.
