O plugin **f5xc-meddpicc** oferece um framework abrangente de
qualificação de vendas e execução de negócios MEDDPICC para vendas
B2B empresariais complexas. Ele transforma o MEDDPICC de um exercício
de checklist em um sistema vivo de execução de negócios com
qualificação baseada em evidências, revisões estruturadas de negócios
e coaching acionável.

<Badge text="v1.0.4" variant="note" /> <Badge text="Productivity" variant="tip" />

## Instalação

```
/plugin install f5xc-meddpicc@f5xc-salesdemos-marketplace
```

## O que é MEDDPICC?

MEDDPICC é um framework de qualificação e execução de negócios
projetado para vendas B2B complexas onde múltiplos stakeholders,
riscos técnicos e etapas de procurement podem comprometer o andamento.

| Letra | Elemento | Propósito |
| ------ | ------- | ------- |
| **M** | Metrics (Métricas) | Resultados de negócio quantificados que o cliente espera |
| **E** | Economic Buyer (Comprador Econômico) | Indivíduo com autoridade final sobre o orçamento |
| **D** | Decision Criteria (Critérios de Decisão) | Requisitos usados para avaliar opções |
| **D** | Decision Process (Processo de Decisão) | Caminho passo a passo da avaliação até o acordo |
| **P** | Paper Process (Processo Burocrático) | Mecânicas de procurement (jurídico, segurança, assinatura) |
| **I** | Identify Pain (Identificar a Dor) | Dor de negócio convincente que cria urgência |
| **C** | Champion (Campeão) | Defensor interno influente |
| **C** | Competition (Competição) | Todas as alternativas (fornecedores, desenvolvimento interno, não fazer nada) |

## Habilidades

### meddpicc-coach

Coaching abrangente da metodologia MEDDPICC que é ativado
automaticamente quando você discute qualificação de negócios, saúde
do pipeline, precisão de forecast ou qualquer elemento do MEDDPICC.
Fornece orientação baseada em evidências, recomendações de ritmo
operacional e links para material de referência detalhado.

<Aside type="tip" title="Ativação automática">
  Esta habilidade é ativada automaticamente quando relevante — nenhum
  comando de barra é necessário.
</Aside>

### deal-qualification

Produz um scorecard MEDDPICC pontuado (0&ndash;24) com análise de
lacunas, avaliação de riscos e recomendação de forecast. Cada
elemento é pontuado de 0&ndash;3 com base em evidências verificadas.

### deal-review

Facilita inspeções semanais estruturadas de negócios com
rastreamento de delta de evidências, identificação de riscos e itens
de ação com responsáveis nomeados e datas de vencimento.

### mutual-action-plan

Constrói um Plano de Ação Mútuo voltado para o cliente e vinculado
aos elementos do MEDDPICC. Inclui marcos de avaliação, roteiro de
procurement, prontidão para implementação e mapeamento de
stakeholders.

### champion-test

Aplica uma avaliação de 5 critérios (Poder, Ganho Pessoal, Acesso,
Inteligência, Ação) com pontuação de 0&ndash;15 para determinar se
seu contato interno é um verdadeiro campeão, campeão emergente,
orientador ou contato amigável.

## Comandos

| Comando | Propósito |
| ------- | ------- |
| `/f5xc-meddpicc:qualify-deal [account]` | Pontuar um negócio com o scorecard MEDDPICC |
| `/f5xc-meddpicc:deal-review [account]` | Facilitar inspeção semanal do negócio |
| `/f5xc-meddpicc:champion-test [contact]` | Avaliar se o contato é um verdadeiro campeão |
| `/f5xc-meddpicc:build-map [account]` | Construir um Plano de Ação Mútuo |

## Agentes

### deal-analyst

Agente de pesquisa somente leitura que analisa a saúde do negócio
revisando documentação local, notas de reunião e planos de conta.
Produz relatórios de avaliação MEDDPICC estruturados com citações de
evidências, análise de lacunas e recomendações de ações priorizadas.

## Biblioteca de Referência

O plugin inclui material de referência abrangente que é carregado
sob demanda:

- **Framework MEDDPICC** &mdash; definições completas e requisitos
  de evidências para todos os 8 elementos
- **Perguntas de Descoberta** &mdash; perguntas testadas em campo
  para cada elemento ao longo do ciclo de negócio
- **Papéis &amp; Responsabilidades** &mdash; matriz de propriedade
  RACI por função da equipe (AE, SE, CSM, PS, Produto, Canal, Deal
  Desk, Jurídico)
- **Gates de Estágio do Negócio** &mdash; o que deve ser verdadeiro
  antes de avançar (da Qualificação Inicial até o Fechamento)
- **Anti-Padrões** &mdash; 10 modos de falha comuns com remediação
  baseada em evidências
- **Rubrica de Pontuação** &mdash; critérios objetivos de pontuação
  0&ndash;3 por elemento
- **Templates de Revisão** &mdash; conjuntos de perguntas para
  inspeção semanal com indicadores de alerta
- **Templates de MAP** &mdash; estrutura de plano de ação voltado
  para o cliente com roteiros de avaliação, procurement e
  implementação
