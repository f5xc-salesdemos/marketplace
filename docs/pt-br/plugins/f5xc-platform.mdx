O plugin **f5xc-platform** fornece automação completa para a
plataforma F5 Distributed Cloud. Ele abrange operações do console
web via Chrome DevTools MCP, operações de API REST com
reconhecimento de especificação via cURL, autenticação
multi-provedor e análise de configuração em 98 tipos de recursos
em 38 domínios.

<Badge text="v3.1.1" variant="note" /> <Badge text="Produtividade" variant="tip" />

## Instalação

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## Habilidades

### api-operations

Operações CRUD de API com reconhecimento de especificação,
cobrindo 98 tipos de recursos em 38 domínios. Fornece endpoints
precisos, modelos de payload, ordenação de dependências e
composições de workflows multi-etapas para load balancers HTTP,
origin pools, políticas WAF, zonas DNS, certificados, cloud sites
e muito mais.

### config-analysis

Capacidade de análise e consultoria de configuração. Analisa
configurações JSON de clientes para responder perguntas sobre
postura de segurança, habilitação de funcionalidades, configurações
de modo, exclusões e melhores práticas. Suporta Q&A multi-turno
com contexto comprimido entre despachos.

### api-auth

Autenticação de API com suporte a autenticação por token de API
via cabeçalho e autenticação por certificado P12 com procedimentos
de validação de token.

### console-auth

Autenticação multi-provedor para o console web do F5 XC.
Detecta automaticamente o tipo de login: login nativo do F5 XC,
SSO Azure com sessão em cache ou SSO Azure com MFA por push
verificado via DUO.

### console-navigator

Navegação determinística para seções do console do F5 XC. Traduz
nomes de seções para caminhos de URL, verifica a saúde da sessão
e confirma o carregamento da página.

### platform-index / api-index / console-index

Roteadores de intenção que classificam solicitações de usuários e
delegam para a habilidade e agente apropriados. Estes são ativados
automaticamente e não são invocáveis pelo usuário.

## Comandos

### /check-api-token

```
/f5xc-platform:check-api-token
```

Valida seu token de API do F5 XC delegando ao agente api-operator.

### /login-console

```
/f5xc-platform:login-console
```

Autentica no console do F5 Distributed Cloud por meio do agente
console-operator.

## Agentes

### api-operator

Agente autônomo de API REST que executa sequências de cURL e jq
para CRUD de recursos, validação de tokens e operações de
configuração. Executa em um subagente para manter payloads JSON
verbosos de API fora do contexto da sessão principal.

### console-operator

Agente autônomo de automação de navegador que executa sequências
de ferramentas Chrome DevTools MCP para autenticação, navegação e
interações com formulários. Executa em um subagente para manter
snapshots de navegador com uso intensivo de tokens fora da sessão
principal.

### config-analyzer

Agente de análise de configuração somente leitura. Analisa
configurações JSON de clientes contra arquivos de referência de
perfil de recursos para responder perguntas sobre postura de
segurança, habilitação de funcionalidades, transições de modo e
melhores práticas.

<Aside type="caution">
  O agente config-analyzer é **efêmero** &mdash; instanciado do
  zero para cada pergunta, sem memória entre despachos. Perguntas
  de acompanhamento devem incluir as descobertas anteriores
  comprimidas no prompt de despacho.
</Aside>
