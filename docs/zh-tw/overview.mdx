**Claude Code 外掛市集**是一個精選的外掛集合，用於擴展 Claude Code 的新技能、命令和代理。市集讓團隊可以共享工具，使每位開發者無需手動設定即可獲得相同的功能。

**f5xc-salesdemos 市集**提供為 [f5xc-salesdemos](https://github.com/f5xc-salesdemos) 文件平台打造的外掛 — 這是一個包含 20 多個儲存庫的生態系統，使用共享建置管線將 Astro/Starlight 網站發布到 GitHub Pages。

## 運作方式

1. **將市集新增**至 Claude Code — 這會將其註冊為外掛來源
2. **從市集安裝外掛** — 每個外掛都會為您的 Claude Code 工作階段新增技能和命令
3. **使用工具** — 外掛會根據情境自動啟用，或者您也可以直接呼叫其命令

## 可用外掛

| 外掛 | 類別 | 說明 |
|--------|----------|-------------|
| [f5xc-docs-tools](/marketplace/plugins/f5xc-docs-tools/) | 生產力 | 用於文件儲存庫的 MDX 內容驗證和審查工具 |
| [f5xc-sales-engineer](/marketplace/plugins/f5xc-sales-engineer/) | 生產力 | 銷售工程師角色框架 — 示範執行、簡報、問答、環境管理 |

## 快速開始

新增市集並安裝文件工具外掛：

```
/plugin marketplace add f5xc-salesdemos/marketplace
/plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
```

請參閱[入門指南](/marketplace/getting-started/)以取得完整的操作說明。

## 架構

市集是一個包含一個或多個外掛的 Git 儲存庫。每個外掛將技能、命令和代理打包為單一可安裝的單元。

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **市集** — 註冊中心。列出可用的外掛及其用於探索和安裝的中繼資料。
- **外掛** — 可安裝的單元。包含一個或多個技能、命令或代理。
- **技能** — 情境知識，Claude 會根據當前任務自動啟用。
- **命令** — 使用者直接呼叫的斜線命令（例如 `/review-mdx`）。
- **代理** — 專門的子代理，可自主處理複雜的多步驟任務。
