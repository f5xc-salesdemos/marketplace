f5xc-salesdemos 市場提供以下外掛程式。
每個外掛程式為 Claude Code 新增技能、指令和代理，
協助處理文件撰寫、示範工作流程、銷售方法論、
品牌規範及開發維運等工作。

## 外掛程式包含什麼？

每個外掛程式可包含：

- **技能** — 當相關情境出現時，Claude 會自動啟用的
  情境知識（例如審查 MDX 檔案）
- **指令** — 您可直接呼叫的斜線指令
  （例如 `/f5xc-docs-tools:review-mdx`）
- **代理** — 用於處理複雜任務的專門子代理
