यह मार्गदर्शिका आपको Claude Code में f5xc-salesdemos
मार्केटप्लेस जोड़ने और अपना पहला प्लगइन इंस्टॉल करने की प्रक्रिया से परिचित कराती है।

## पूर्वापेक्षाएँ

- Claude Code **v1.0.33** या बाद का संस्करण (प्लगइन मार्केटप्लेस समर्थन)
- [f5xc-salesdemos](https://github.com/f5xc-salesdemos)
  संगठन तक पहुँच वाला एक GitHub खाता

## इंस्टॉलेशन

<Steps>

1. **मार्केटप्लेस जोड़ें**

   किसी भी Claude Code सत्र में, यह कमांड चलाएँ:

   ```
   /plugin marketplace add f5xc-salesdemos/marketplace
   ```

   यह मार्केटप्लेस को एक प्लगइन स्रोत के रूप में पंजीकृत करता है। आपको
   यह केवल एक बार करना होगा।

2. **उपलब्ध प्लगइन ब्राउज़ करें**

   मार्केटप्लेस में उपलब्ध प्लगइन की सूची देखें:

   ```
   /plugin marketplace list f5xc-salesdemos-marketplace
   ```

3. **प्लगइन इंस्टॉल करें**

   अपनी आवश्यकता का प्लगइन इंस्टॉल करें:

   ```
   /plugin install f5xc-docs-tools@f5xc-salesdemos-marketplace
   /plugin install f5xc-sales-engineer@f5xc-salesdemos-marketplace
   ```

4. **प्लगइन का उपयोग करें**

   किसी भी f5xc-salesdemos कंटेंट रिपॉज़िटरी में नेविगेट करें।

   **डॉक्स टूल्स** के लिए, MDX समीक्षा कमांड चलाएँ:

   ```
   /f5xc-docs-tools:review-mdx
   ```

   **सेल्स इंजीनियर** के लिए, ट्रिगर वाक्यांशों का उपयोग करें जैसे
   "prepare the demo", "run the demo", या "walk through
   the demo" — प्लगइन स्वचालित रूप से सक्रिय हो जाता है।

</Steps>

## प्लगइन अपडेट करना

इंस्टॉल किए गए प्लगइन को नवीनतम संस्करण में अपडेट करने के लिए:

```
/plugin update f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin update f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

## अनइंस्टॉल करना

प्लगइन हटाएँ:

```
/plugin uninstall f5xc-docs-tools@f5xc-salesdemos-marketplace
/plugin uninstall f5xc-sales-engineer@f5xc-salesdemos-marketplace
```

मार्केटप्लेस को पूरी तरह हटाएँ:

```
/plugin marketplace remove f5xc-salesdemos-marketplace
```

## टीम कॉन्फ़िगरेशन

अपनी पूरी टीम के लिए मार्केटप्लेस को पूर्व-कॉन्फ़िगर करने के लिए, इसे
अपने प्रोजेक्ट की `.claude/settings.json` फ़ाइल में जोड़ें:

```json
{
  "extraKnownMarketplaces": [
    "f5xc-salesdemos/marketplace"
  ],
  "enabledPlugins": [
    "f5xc-docs-tools@f5xc-salesdemos-marketplace",
    "f5xc-sales-engineer@f5xc-salesdemos-marketplace"
  ]
}
```

<Aside type="tip" title="साझा सेटिंग्स">
  `.claude/settings.json` को अपनी रिपॉज़िटरी में कमिट करने का मतलब है कि
  हर टीम सदस्य को मार्केटप्लेस और प्लगइन
  स्वचालित रूप से मिल जाते हैं — किसी मैनुअल सेटअप की आवश्यकता नहीं।
</Aside>

यह सुनिश्चित करता है कि हर योगदानकर्ता के पास रिपॉज़िटरी में अपने
पहले Claude Code सत्र से ही समान टूलिंग उपलब्ध हो।
