**f5xc-docs-pipeline** प्लगइन f5-sales-demo रिपॉज़िटरीज़ के लिए
डॉक्यूमेंटेशन पाइपलाइन आर्किटेक्चर ज्ञान प्रदान करता है। यह कंटेंट
ऑथरिंग में मार्गदर्शन करता है, मल्टी-रेपो बिल्ड्स में कॉन्फ़िग
ओनरशिप को स्पष्ट करता है, और लोकल Docker-आधारित प्रीव्यू प्रदान करता है।

<Badge text="v1.0.2" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-docs-pipeline@f5-sales-demo-marketplace
```

## स्किल्स

### content-author

कंटेंट ऑथरिंग गाइड जिसमें फ़ाइल संरचना, MDX सिंटैक्स नियम,
फ्रंटमैटर आवश्यकताएँ, इमेज रेफरेंस, और लोकल Docker प्रीव्यू
शामिल है। यह `docs/` कंटेंट बनाते या संपादित करते समय, MDX फ़ाइलों
के साथ काम करते समय, या लोकल प्रीव्यू सेटअप करते समय सक्रिय होता है।

### pipeline-navigator

डॉक्यूमेंटेशन पाइपलाइन आर्किटेक्चर जो बताता है कि चार अपस्ट्रीम
रिपॉज़िटरीज़: docs-theme, docs-builder, docs-control, और docs-icons
में कहाँ बदलाव करने हैं। कॉन्फ़िग ओनरशिप (`astro.config.mjs`,
`content.config.ts`), रिलीज़ डिस्पैच चेन, और मैनेज्ड फ़ाइल ट्रैकिंग
को स्पष्ट करता है।

<Aside type="tip">
  जब आपको यह समझने की आवश्यकता हो कि कोई विशिष्ट कॉन्फ़िगरेशन
  फ़ाइल या कंपोनेंट किस रिपॉज़िटरी का है, तो pipeline-navigator
  का उपयोग करें।
</Aside>

## कमांड्स

### /preview-docs

```
/f5xc-docs-pipeline:preview-docs
```

डॉक्यूमेंटेशन का प्रीव्यू करने के लिए एक लोकल Docker डेव सर्वर
शुरू करता है। लाइव प्रीव्यू के लिए आपकी लोकल `docs/` डायरेक्टरी को
माउंट करके शेयर्ड Docker बिल्ड इमेज चलाता है।
