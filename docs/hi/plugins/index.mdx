f5xc-salesdemos मार्केटप्लेस निम्नलिखित प्लगइन्स प्रदान करता है।
प्रत्येक प्लगइन Claude Code में स्किल्स, कमांड्स और एजेंट्स जोड़ता है
जो डॉक्यूमेंटेशन, डेमो वर्कफ़्लो, सेल्स मेथडोलॉजी,
ब्रांडिंग और डेवलपमेंट ऑपरेशंस में सहायता करते हैं।

## एक प्लगइन में क्या होता है?

प्रत्येक प्लगइन में शामिल हो सकते हैं:

- **स्किल्स** — संदर्भगत ज्ञान जिसे Claude प्रासंगिक होने पर
  स्वचालित रूप से सक्रिय करता है (जैसे, MDX फ़ाइलों की समीक्षा)
- **कमांड्स** — स्लैश कमांड्स जिन्हें आप सीधे इनवोक करते हैं
  (जैसे, `/f5xc-docs-tools:review-mdx`)
- **एजेंट्स** — जटिल कार्यों के लिए विशेष सब-एजेंट्स
