**f5xc-platform** प्लगइन F5 Distributed Cloud प्लेटफ़ॉर्म के लिए
पूर्ण ऑटोमेशन प्रदान करता है। यह Chrome DevTools MCP के माध्यम से
वेब कंसोल ऑपरेशन, cURL के माध्यम से स्पेक-अवेयर REST API ऑपरेशन,
मल्टी-प्रोवाइडर प्रमाणीकरण, और 38 डोमेन में 98 रिसोर्स प्रकारों
के कॉन्फ़िगरेशन विश्लेषण को कवर करता है।

<Badge text="v3.1.1" variant="note" /> <Badge text="Productivity" variant="tip" />

## इंस्टॉलेशन

```
/plugin install f5xc-platform@f5xc-salesdemos-marketplace
```

## स्किल्स

### api-operations

38 डोमेन में 98 रिसोर्स प्रकारों को कवर करने वाले स्पेक-अवेयर
API CRUD ऑपरेशन। HTTP लोड बैलेंसर, ऑरिजिन पूल, WAF पॉलिसी,
DNS ज़ोन, सर्टिफ़िकेट, क्लाउड साइट, और अधिक के लिए सटीक
एंडपॉइंट, पेलोड टेम्पलेट, डिपेंडेंसी ऑर्डरिंग, और मल्टी-स्टेप
वर्कफ़्लो कम्पोज़िशन प्रदान करता है।

### config-analysis

कॉन्फ़िगरेशन विश्लेषण और सलाहकार क्षमता। सुरक्षा स्थिति,
फ़ीचर सक्रियता, मोड सेटिंग्स, अपवर्जन, और सर्वोत्तम प्रथाओं
के बारे में प्रश्नों का उत्तर देने के लिए कस्टमर JSON
कॉन्फ़िगरेशन का विश्लेषण करता है। डिस्पैच के बीच संपीड़ित
संदर्भ के साथ मल्टी-टर्न प्रश्नोत्तर का समर्थन करता है।

### api-auth

API प्रमाणीकरण जो API टोकन हेडर प्रमाणीकरण और टोकन सत्यापन
प्रक्रियाओं के साथ P12 सर्टिफ़िकेट प्रमाणीकरण का समर्थन
करता है।

### console-auth

F5 XC वेब कंसोल के लिए मल्टी-प्रोवाइडर प्रमाणीकरण। लॉगिन
प्रकार को स्वचालित रूप से पहचानता है: नेटिव F5 XC लॉगिन,
कैश्ड सेशन के साथ Azure SSO, या DUO वेरिफ़ाइड पुश MFA के
साथ Azure SSO।

### console-navigator

F5 XC कंसोल सेक्शन में डिटर्मिनिस्टिक नेविगेशन। सेक्शन नामों
को URL पाथ में अनुवादित करता है, सेशन हेल्थ की जाँच करता है,
और पेज लोड की पुष्टि करता है।

### platform-index / api-index / console-index

इंटेंट राउटर जो उपयोगकर्ता अनुरोधों को वर्गीकृत करते हैं और
उपयुक्त स्किल और एजेंट को डेलिगेट करते हैं। ये स्वचालित रूप
से सक्रिय होते हैं और उपयोगकर्ता द्वारा आमंत्रित नहीं किए
जा सकते।

## कमांड

### /check-api-token

```
/f5xc-platform:check-api-token
```

api-operator एजेंट को डेलिगेट करके आपके F5 XC API टोकन को
सत्यापित करता है।

### /login-console

```
/f5xc-platform:login-console
```

console-operator एजेंट के माध्यम से F5 Distributed Cloud
कंसोल में प्रमाणीकरण करता है।

## एजेंट

### api-operator

स्वायत्त REST API एजेंट जो रिसोर्स CRUD, टोकन सत्यापन, और
कॉन्फ़िगरेशन ऑपरेशन के लिए cURL और jq अनुक्रम निष्पादित
करता है। मुख्य सेशन संदर्भ से वर्बोस API JSON पेलोड को बाहर
रखने के लिए सबएजेंट में चलता है।

### console-operator

स्वायत्त ब्राउज़र ऑटोमेशन एजेंट जो प्रमाणीकरण, नेविगेशन,
और फ़ॉर्म इंटरैक्शन के लिए Chrome DevTools MCP टूल अनुक्रम
निष्पादित करता है। मुख्य सेशन से टोकन-भारी ब्राउज़र स्नैपशॉट
को बाहर रखने के लिए सबएजेंट में चलता है।

### config-analyzer

रीड-ओनली कॉन्फ़िगरेशन विश्लेषण एजेंट। सुरक्षा स्थिति, फ़ीचर
सक्रियता, मोड ट्रांज़िशन, और सर्वोत्तम प्रथाओं के बारे में
प्रश्नों का उत्तर देने के लिए रिसोर्स प्रोफ़ाइल संदर्भ
फ़ाइलों के विरुद्ध कस्टमर JSON कॉन्फ़िगरेशन का विश्लेषण
करता है।

<Aside type="caution">
  config-analyzer एजेंट **अल्पकालिक** है &mdash; प्रत्येक प्रश्न
  के लिए नए सिरे से उत्पन्न होता है और डिस्पैच के बीच कोई
  मेमोरी नहीं रखता। अनुवर्ती प्रश्नों में डिस्पैच प्रॉम्प्ट
  में पिछले निष्कर्षों को संपीड़ित रूप में शामिल करना
  आवश्यक है।
</Aside>
