एक **Claude Code प्लगइन मार्केटप्लेस** प्लगइन का एक क्यूरेटेड संग्रह है
जो Claude Code को नई स्किल, कमांड और एजेंट के साथ
विस्तारित करता है। मार्केटप्लेस टीमों को टूलिंग साझा करने देते हैं ताकि
प्रत्येक डेवलपर को मैन्युअल सेटअप के बिना समान क्षमताएं मिलें।

**f5-sales-demo मार्केटप्लेस** [f5-sales-demo](https://github.com/f5-sales-demo)
डॉक्यूमेंटेशन प्लेटफ़ॉर्म के लिए बनाए गए प्लगइन प्रदान करता है — 20+
रिपॉज़िटरी का एक इकोसिस्टम जो एक साझा बिल्ड पाइपलाइन
का उपयोग करके GitHub Pages पर Astro/Starlight साइटें प्रकाशित करती हैं।

## यह कैसे काम करता है

1. **मार्केटप्लेस जोड़ें** Claude Code में — यह इसे एक प्लगइन
   स्रोत के रूप में पंजीकृत करता है
2. **मार्केटप्लेस से प्लगइन इंस्टॉल करें** — प्रत्येक प्लगइन आपके
   Claude Code सत्र में स्किल और कमांड जोड़ता है
3. **टूल्स का उपयोग करें** — प्लगइन संदर्भ के आधार पर स्वचालित
   रूप से सक्रिय होते हैं, या आप उनके कमांड सीधे इनवोक करते हैं

## उपलब्ध प्लगइन

## त्वरित शुरुआत

मार्केटप्लेस जोड़ें और डॉक्स टूल्स प्लगइन इंस्टॉल करें:

```
/plugin marketplace add f5-sales-demo/marketplace
/plugin install f5xc-docs-tools@f5-sales-demo-marketplace
```

पूर्ण वॉकथ्रू के लिए [शुरुआत करना](/marketplace/getting-started/)
देखें।

## आर्किटेक्चर

एक मार्केटप्लेस एक Git रिपॉज़िटरी है जिसमें एक या अधिक
प्लगइन होते हैं। प्रत्येक प्लगइन स्किल, कमांड और एजेंट को
एक ही इंस्टॉल करने योग्य इकाई में पैकेज करता है।

```
Marketplace (marketplace.json)
└── Plugin (plugin.json)
    ├── Skills (SKILL.md)
    │   └── References (data files)
    ├── Commands (.md)
    └── Agents (.md)
```

- **मार्केटप्लेस** — रजिस्ट्री। खोज और इंस्टॉलेशन के लिए
  मेटाडेटा के साथ उपलब्ध प्लगइन सूचीबद्ध करती है।
- **प्लगइन** — इंस्टॉल करने योग्य इकाई। एक या अधिक
  स्किल, कमांड, या एजेंट शामिल होते हैं।
- **स्किल** — संदर्भगत ज्ञान जिसे Claude कार्य के आधार
  पर स्वचालित रूप से सक्रिय करता है।
- **कमांड** — एक स्लैश कमांड जिसे उपयोगकर्ता सीधे
  इनवोक करता है (जैसे, `/review-mdx`)।
- **एजेंट** — एक विशेष सब-एजेंट जो जटिल,
  बहु-चरणीय कार्यों को स्वायत्त रूप से संभालता है।
