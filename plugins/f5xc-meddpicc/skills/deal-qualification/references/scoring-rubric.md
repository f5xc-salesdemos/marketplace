# MEDDPICC Scoring Rubric

Detailed criteria for scoring each element 0–3.

---

## Metrics (M)

| Score | Criteria |
| ----- | -------- |
| 3 | Customer-stated baseline AND target with measurement method and timeline. Metrics tied to executive priority and budget justification. |
| 2 | Baseline or target identified but not both. Measurement method unclear. General agreement on business impact. |
| 1 | Qualitative statements only ("improve security", "reduce costs"). No baseline or measurement method. Seller-hypothesized metrics. |
| 0 | No metrics discussion has occurred. |

---

## Economic Buyer (E)

| Score | Criteria |
| ----- | -------- |
| 3 | Named individual with confirmed budget authority. Direct access achieved (meeting held or scheduled). EB priorities and personal win documented. EB is actively sponsoring. |
| 2 | EB identified by name and title. No direct access yet but plan exists. Champion confirms EB's priorities. |
| 1 | Role identified but not the specific person. Access only through intermediaries. Assumptions about budget authority. |
| 0 | Unknown who controls budget. No plan to identify or access. |

---

## Decision Criteria (D)

| Score | Criteria |
| ----- | -------- |
| 3 | Written criteria from customer with priority ranking. Your competitive position mapped per criterion. Criteria influenced toward your strengths. Key stakeholders aligned on criteria. |
| 2 | Criteria partially documented. Priority ranking informal. Some influence actions taken. Not all stakeholders aligned. |
| 1 | General requirements known but not documented or ranked. No influence actions taken. Single contact's view only. |
| 0 | Don't know how the customer will evaluate solutions. |

---

## Decision Process (D)

| Score | Criteria |
| ----- | -------- |
| 3 | Step-by-step process documented with named owners, dates, and required artifacts. Legal/security/procurement flow mapped. No surprises expected. |
| 2 | Major steps identified but dates or owners missing. Some gates unclear. General timeline known. |
| 1 | High-level understanding only ("they'll do a POV then decide"). No dates or named owners. |
| 0 | Unknown how the customer will make and approve the decision. |

---

## Paper Process (P)

| Score | Criteria |
| ----- | -------- |
| 3 | Procurement steps documented with owners and timelines. Security artifacts submitted. Legal terms in review. Quote/PO format confirmed. Signature workflow mapped. |
| 2 | Some procurement steps known. Security requirements identified but artifacts not yet submitted. Legal terms not yet reviewed. |
| 1 | Aware that procurement exists but steps unknown. No artifacts staged. Assuming it will be "straightforward." |
| 0 | No procurement discovery done. |

---

## Identify Pain (I)

| Score | Criteria |
| ----- | -------- |
| 3 | Pain quantified with business impact. Consequence of inaction documented. Triggering event identified. Multiple stakeholders confirm the pain. Clear urgency driver. |
| 2 | Pain acknowledged by customer but not fully quantified. Single stakeholder confirmation. General urgency without specific driver. |
| 1 | Pain implied but not explicitly confirmed by customer. Seller's interpretation of the problem. No urgency driver. |
| 0 | No pain identified or discussed. Solution looking for a problem. |

---

## Champion (C)

| Score | Criteria |
| ----- | -------- |
| 3 | Named individual with influence and credibility. Personal win documented. Has taken concrete action this week (EB intro, internal advocacy, shared intel). Actively coaching you. |
| 2 | Friendly contact with some influence. Willing to help but hasn't taken significant action yet. Personal win unclear. |
| 1 | Friendly contact without organizational power. Passive supporter. No concrete actions taken. |
| 0 | No internal advocate identified. |

---

## Competition (C)

| Score | Criteria |
| ----- | -------- |
| 3 | Competitors named and qualified. Customer's perception of each alternative documented. Differentiation strategy clear. Competitive proof planned or delivered. "Do nothing" risk assessed. |
| 2 | Competitors identified but customer's perception unclear. General differentiation known. No competitive proof planned. |
| 1 | Assumptions about competition. "We don't think they're talking to anyone." No differentiation strategy. |
| 0 | No competitive intelligence gathered. |
