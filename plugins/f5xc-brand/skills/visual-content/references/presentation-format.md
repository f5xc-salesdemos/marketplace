### Slide types

#### Title slide

- Background: F5 Red or River gradient (135deg)
- Title: white, Neusa Bold, 36-44pt
- Subtitle: white, Proxima Regular, 18-24pt
- F5 logo: bottom-right, white variant

#### Section divider

- Background: River (`#0e41aa`)
- Section title: white, Neusa Bold, 32-40pt
- F5 logo: bottom-right, white

#### Content slide

- Background: white
- Title: black, Neusa Bold, 28-32pt
- Body: Black-3, Proxima Regular, 16-18pt
- F5 logo: top-right, red variant, small

#### Data / chart slide

- Background: white
- Chart colors in order: River, Bay, Jade, Tangerine,
  Raspberry, Eggplant (provides 6 distinct categories)
- Labels: Black-3, Proxima Regular
- Gridlines: White-3 (`#e6e6e6`)

#### Quote slide

- Background: white or River-4 (`#072155`)
- Quote text: Neusa Next Pro Wide Italic, 24-32pt
- Attribution: Proxima Regular, 14-16pt
- Left accent bar: F5 Red, 4px wide

### Logo placement

- Every slide should include the F5 logo
- Recommended positions: top-right or bottom-right
- Size: small (approximately 1 inch / 72pt wide)
- Use red variant on white backgrounds, white variant
  on dark/colored backgrounds
- Maintain clear space — do not crowd with other elements

## Video content

### YouTube thumbnails

| Property | Value |
| -------- | ----- |
| Dimensions | 1280x720px (16:9) |
| Background | F5 Red gradient, River gradient, or dark navy (River-4) |
| Text font | Neusa Next Pro Wide Bold (or Arial Black) |
| Text color | White |
| Text size | Large enough to read at 120x68px preview |
| F5 logo | Small watermark in corner |
| Max text | 5-7 words — communicate at a glance |

### Title cards (video intro)

| Property | Value |
| -------- | ----- |
| Background | River-4 (`#072155`) or F5 Red gradient |
| Title font | Neusa Next Pro Wide Bold, 48-64pt |
| Title color | White |
| Subtitle | Proxima Nova Regular, 24-32pt, white |
| F5 logo | Center or bottom-right |
| Duration | 3-5 seconds |

### Lower thirds

| Property | Value |
| -------- | ----- |
| Background | Semi-transparent dark overlay or solid River-4 |
| Name font | Neusa Next Pro Wide Bold, 24pt, white |
| Title font | Proxima Nova Regular, 18pt, White-3 |
| Accent bar | F5 Red, 3-4px, left edge |
| Position | Bottom-left of frame |

## Social media

### Platform image dimensions

| Platform | Format | Dimensions | Aspect |
| -------- | ------ | ---------- | ------ |
| LinkedIn shared link | Card | 1200x630 | 1.91:1 |
| LinkedIn post image | Square | 1200x1200 | 1:1 |
| X (Twitter) card | Card | 1200x630 | 1.91:1 |
| X post image | Landscape | 1600x900 | 16:9 |
| Instagram post | Square | 1080x1080 | 1:1 |
| Instagram story | Portrait | 1080x1920 | 9:16 |
| Facebook shared link | Card | 1200x630 | 1.91:1 |
| YouTube thumbnail | Landscape | 1280x720 | 16:9 |

### Social image design

- Background: brand color (River, Red, or River-4)
  with contrasting white text
- Text: Neusa Next Pro Wide Bold (or Arial Black)
- Keep text to 3-5 words — social images must
  communicate instantly
- Include F5 logo
- Use high contrast — social feeds are visually noisy
