# f5xc-brand

F5 corporate branding enforcement for content creation
across all media types.

## What it does

This plugin enforces F5 corporate brand guidelines when
creating any content — documentation, diagrams,
presentations, video, social media, or visual assets. It
references the canonical brand specs in `docs-theme` and
`docs-icons` rather than duplicating them.

## Skills

| Skill | Purpose |
| ----- | ------- |
| `brand-guardian` | Core brand enforcement — colors, typography, logos, icons, tone, accessibility |
| `visual-content` | Format-specific rules for diagrams, presentations, screenshots, video |
| `brand-reviewer` | On-demand brand compliance audit with structured report |

## Commands

| Command | Purpose |
| ------- | ------- |
| `/review-brand` | Audit files for F5 brand compliance |

## Source of truth

This plugin does not contain the brand specs themselves.
It points to:

- **docs-theme** — colors, typography, logos, design
  tokens, CSS
- **docs-icons** — icon packages (f5-brand, f5xc, 10
  third-party packs)
